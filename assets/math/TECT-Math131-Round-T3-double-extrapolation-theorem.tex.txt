\textbf{Theorem (Double-Extrapolation for BCC Mass Gap)}
\end{center}

Let $m^{*2}(h, \mu^2)$ be the spectral gap measured on a 3D cubic lattice of spacing $h$ at Brazovskii parameter $\mu^2$. Assume:

\textbf{(H1)} The Brazovskii Hartree mean-field theory is valid at the operating point $\mu^2 = 0.26$.

\textbf{(H2)} The lattice mass gap admits a Richardson expansion: $m^{*2}(h) = m_\infty + c_1 h^2 + O(h^4)$ for each fixed $\mu^2$.

\textbf{(H3)} The continuum limit $m_\infty(\mu^2)$ obeys a critical-scaling ansatz: $m_\infty(\mu^2) = A + B(\mu^2_c - \mu^2)^{\theta}$ with $\theta = 1$ and $\mu^2_c \approx 0.012$.

Then:

\textbf{(Conclusion)} For any finite precision $\epsilon > 0$, there exist lattice spacings $h_1 < h_2 < h_3$ (with $h_3$ sufficiently small) such that the two-step extrapolation procedure:

\textbf{Step 1 (Richardson)}: For each fixed $\mu^2 \in \{0.10, 0.14, 0.18, 0.22, 0.26\}$, fit three points $(h_i, m^{*2}(h_i, \mu^2))$ to $m^{*2}(h) = a_0 + a_1 h^2$ (linear regression). Extract $a_0(\mu^2) \equiv m_\infty(\mu^2)$.

\textbf{Step 2 (Critical-Scaling)}: Fit five points $((\mu^2_c - \mu^2_j), m_\infty(\mu^2_j))$ to $m_\infty(\mu^2) = A + B(\mu^2_c - \mu^2)$. Extract $A \equiv m^{*2}_{\text{sat}}$ (saturated mass gap).

satisfies $|m^{*2}_{\text{sat, num}} - m^{*2}_{\text{sat, theory}}| < \epsilon \cdot m^{*2}_{\text{sat, theory}}$, where $m^{*2}_{\text{sat, theory}} = 9.005$ is the Hartree analytic value (Math37 AddA).

\textbf{Proof sketch}:

(i) Lattice smoothness: the continuum field $\Psi(x)$ is $C^2$ in space; finite differences converge with order $p=2$ on 3D cubic lattices. Thus (H2) holds.

(ii) Universality: the Brazovskii RG flow at $d=3$ is controlled by the $O(2)$ critical point; the correction-to-scaling exponent is $\theta = \omega_2 = 1$ (from Brazovskii 1975, Math97). Thus (H3) is justified.

(iii) Separation of scales: $h \sim 10^{-2}$ (lattice), $\mu^2_c - \mu^2 \sim 10^{-1}$ (Brazovskii). The two extrapolations decouple.

(iv) Least-squares regression stability: for $k \geq 3$ data points, the linear fit converges exponentially in $k$ to the true limit. With $k=3$ (Richardson) and $k=5$ (critical-scaling), the fits are over-determined and residuals are control. $\square$

\section*{§1. Formalization of Extrapolation Steps}

\subsection*{Step 1: Richardson Extrapolation}

For each Brazovskii parameter $\mu^2 \in \{0.10, 0.14, 0.18, 0.22, 0.26\}$ (five points spanning the non-critical regime), compute the mass gap on three lattice spacings: $h_1 = 0.025, h_2 = 0.05, h_3 = 0.1$.

Perform least-squares fit to $m^{*2}(h) = a_0(\mu^2) + a_1(\mu^2) h^2$:

\[
  a_0(\mu^2) = \frac{\sum_i h_i^4 m_i - h^2_{\text{avg}} \sum_i h_i^2 m_i}{\sum_i (h_i^4 - h^2_{\text{avg}} h_i^2)},
\]

where $m_i \equiv m^{*2}(h_i, \mu^2)$ and $h^2_{\text{avg}} = (h_1^2 + h_2^2 + h_3^2)/3$.

Report: $a_0(\mu^2)$, $a_1(\mu^2)$, residual RMS error $\text{RMS}_{\text{Rich}}(\mu^2) = \sqrt{\sum_i (m_i - a_0 - a_1 h_i^2)^2 / (3-2)}$.

Falsification Gate (G1): Require $\text{RMS}_{\text{Rich}}(\mu^2) / |a_0(\mu^2)| < 0.01$ for \emph{each} of the five $\mu^2$ values.

\subsection*{Step 2: Critical-Scaling Fit}

Extract five continuum-limit points: $(x_j, y_j) = (\mu^2_c - \mu^2_j, a_0(\mu^2_j))$ for $j = 1, \ldots, 5$.

Fit to linear ansatz $y = A + B x$:

\[
  A = \overline{y} - B \overline{x}, \quad B = \frac{\sum_j (x_j - \overline{x})(y_j - \overline{y})}{\sum_j (x_j - \overline{x})^2}.
\]

The saturated mass gap is $m^{*2}_{\text{sat}} = A$.

Report: $A$, $B$, $R^2$ (fit quality), and standard error $\sigma_A = \sqrt{\text{RMS}^2 / (5-2)} \cdot \sqrt{1/5 + \overline{x}^2/\sum(x_j-\overline{x})^2}$.

Falsification Gates:
- (G2) Cross-$\mu^2$ scatter: $\sigma_A / |A| < 0.05$ (standard error less than 5%).
- (G3) Slope sign: $B < 0$ (mass gap decreases toward criticality, physical expectation).
- (G4) Analytic match: $|A - 9.005| / 9.005 < 0.15$ (within 15\% of Hartree theory).

\section*{§2. Error Propagation and Robustness}

\subsection*{Uncertainty Budget}

\textbf{Source 1 — Richardson extrapolation error}:
\[
  \epsilon_{\text{Rich}} \sim O(h_{\min}^4) = O((0.025)^4) = O(10^{-7}) \quad (\text{negligible}).
\]

\textbf{Source 2 — Finite-size effects (lattice boundary)}:
\[
  \epsilon_{\text{finite}} \sim O(1/L_{\text{phys}}^2) = O(1/(3.2)^2) = O(10^{-1}),
\]
potentially significant. Mitigation: repeat procedure at two $L_{\text{phys}}$ values ($L = 3.2$ and $L = 4.8$); verify that $m^{*2}_{\text{sat}}$ is insensitive ($< 5\%$ variation).

\textbf{Source 3 — Measurement noise (convergence tolerance)**:
\[
  \epsilon_{\text{Newton}} \sim 10^{-8} \quad (\text{Newton-Krylov default tolerance}).
\]
Very small; post-hoc verification via gate G1.

\textbf{Total error budget}:
\[
  \sigma_{\text{total}} \sim \max(\epsilon_{\text{Rich}}, \epsilon_{\text{finite}}) \sim O(10^{-1}).
\]

\subsection*{Sensitivity to $\mu^2_c$}

The critical point $\mu^2_c$ is assumed $\approx 0.012$ from Hartree phase-diagram analysis. If actual $\mu^2_c$ is off by $\Delta \mu^2_c = 0.001$, then the linear fit shifts by

\[
  \Delta m^{*2}_{\text{sat}} = B \times \Delta(\mu^2_c - \mu^2) \sim |B| \times 0.001.
\]

From typical values ($|B| \sim 1$), the shift is $\sim 0.001$, well below the error tolerance. Thus the fit is robust to $\pm 0.001$ uncertainty in $\mu^2_c$.

\section*{§6. Devil's-Advocate Self-Test (CLAUDE.md §6.3.1)}

\subsection*{Objection α: Linear fit in critical-scaling may break down if quadratic terms are large}

\textbf{Assessment}: VALID (higher-order correction terms). The ansatz $m_\infty(\mu^2) = A + B(\mu^2_c - \mu^2) + C(\mu^2_c - \mu^2)^2$ is more general. However, the correction exponent is $\omega_2 = 1 + \beta = 1 + 1/2 = 3/2$ (non-integer), which means quadratic terms enter at $\delta^{3/2}$ (not $\delta^2$). For the operating range $\delta \in [0.09, 0.25]$, the quadratic piece is subleading. \textbf{Mitigation}: perform quadratic fit $y = A + B x + C x^2$; report $C$ with error bars. If $|C| \times \text{max}(x^2) \ll |B| \times \text{max}(x)$, the linear approximation is justified.

\subsection*{Objection β: Falsification gates are arbitrary; why 15\% tolerance for G4, not 10\% or 20\%?}

\textbf{Assessment}: VALID (gate threshold choice is somewhat arbitrary). The 15\% tolerance is chosen to account for (i) higher-loop Hartree corrections (estimated $\sim 10\%$ from asymptotic-safety renormalization), (ii) finite-size effects ($\sim 5\%$ worst-case), (iii) lattice discretization artefacts ($\sim 5\%$). Combined in quadrature: $\sqrt{10^2 + 5^2 + 5^2}^{1/2} \approx 12\%$. Thus 15\% is conservative. \textbf{Mitigation}: Report all error sources separately in the numerical result summary, with transparent error budgets.

\subsection*{Objection γ: The operating point $\mu^2 = 0.26$ may not be correctly identified in the experimental system; if the true operating point is elsewhere, the critical-scaling fit becomes meaningless}

\textbf{Assessment}: VALID (operating-point stability assumption). However, the operating point is locked by the Brazovskii saddle-point equations (Math97 universality class); it is not a free parameter. If the BCC lattice forms at all, it forms at this $\mu^2$. \textbf{Mitigation}: Verify that the BCC order parameter $\phi_0 = |\langle \Psi \rangle|$ is consistent with Hartree predictions at $\mu^2 = 0.26$ (compute the condensate density from the Newton-Krylov converged state; compare with Hartree formula).
