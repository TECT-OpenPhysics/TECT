\section*{Math82-Addendum-I: Cold-start scan runbook and pre-registered Brazovskii prediction}

\subsection*{1. Runbook — exact shell commands for 7 independent cold-start Newton iterations}

\paragraph*{Context.} Math82-G §3 Regime III claimed that the subset-4-cosine branch
``terminates below $\mu^2 \approx -0.1$''. The Math82-G2 audit (2026-04-24) reclassified
this claim as \textbf{undetermined} because Points 4-7 inherited a near-trivial-vacuum
warm-start from Point 3 ($\Psi^* \approx 0$, $F(\Psi^*) \approx -3.31\times 10^{-10}$).
A warm-start Newton chain cannot determine whether a truly non-trivial subset-4 equilibrium
exists at deeper $\mu^2 < 0$. This addendum provides the direct experimental protocol:
independent, zero-knowledge cold-start Newton from a fresh subset-4-cosine seed at each
of the 7 $\mu^2$ values.

\paragraph*{Prerequisites.}
\begin{itemize}
\item PyTorch + NumPy + SciPy available (user's local environment; sandbox
  unavailable in this session).
\item `Codes/pde/bcc_analytic_seed.py` (v0.2+) and `Codes/pde/continuation_mu2_v25.py` (v2.5+) checked out.
\item Grid size $N=32$, BCC box $L=62.20036$ (from Math82-G2, Math82-F).
\item Newton tolerances: $\mathrm{tol}_{\rm Newton}=10^{-8}$, $\mathrm{max\_Newton}=15$.
\item Brazovskii locked parameters: $\lambda=-0.43$, $\gamma=1.62$.
\end{itemize}

\paragraph*{Stage 1: Generate 7 fresh subset-4-cosine seeds with random phases.}

Execute these commands in the bash shell at the repository root (`C:\Dev\TECT2\Contents`).
Random phases break the deterministic warm-start collapse pathology.

\begin{verbatim}
#!/bin/bash
# Math82-I Stage 1: generate fresh subset-4-cosine seeds

export PYTHONPATH=.
OUTDIR="Runs/math82i_cold_start_seeds_$(date +%Y%m%d_%H%M%S)"
mkdir -p "$OUTDIR"

# 7 independent μ² values
declare -a MU2_VALUES=("+5e-3" "-0.02" "-0.1" "-0.5" "-0.7" "-0.85" "-1.0")

for i in "${!MU2_VALUES[@]}"; do
  MU2="${MU2_VALUES[$i]}"
  SEED_FILE="$OUTDIR/psi_seed_subset4_mu2_${MU2}_idx${i}.npy"

  echo "[Stage1-Seed $((i+1))/7] Generating subset-4-cosine seed at μ²=$MU2, index=$i"

  python Codes/pde/bcc_analytic_seed.py \
    --mode subset_4cosine_random_phase \
    --N 32 \
    --L 62.20036 \
    --seed_idx "$i" \
    --output "$SEED_FILE" \
    --verbose

  if [ ! -f "$SEED_FILE" ]; then
    echo "ERROR: seed generation failed for index $i, μ²=$MU2"
    exit 1
  fi
done

echo "Stage 1 complete. Seeds saved to: $OUTDIR"
echo "Seed directory: $OUTDIR"
\end{verbatim}

\paragraph*{Stage 2: Run 7 independent single-point Newton solves.}

For each μ² value, invoke `continuation_mu2_v25.py` with \textbf{no continuation chain}
(single-point Newton from the cold-start seed). Trust-region Newton terminates when
convergence threshold is reached or max\_Newton iterations are exhausted.

\begin{verbatim}
#!/bin/bash
# Math82-I Stage 2: independent cold-start Newton solves

export PYTHONPATH=.
SEEDS_DIR="${1:?Usage: $0 <seeds_directory>}"
RESULTS_DIR="Runs/math82i_cold_start_results_$(date +%Y%m%d_%H%M%S)"
mkdir -p "$RESULTS_DIR"

# 7 μ² values (same order as seed generation)
declare -a MU2_VALUES=("+5e-3" "-0.02" "-0.1" "-0.5" "-0.7" "-0.85" "-1.0")

for i in "${!MU2_VALUES[@]}"; do
  MU2="${MU2_VALUES[$i]}"
  SEED_FILE="$SEEDS_DIR/psi_seed_subset4_mu2_${MU2}_idx${i}.npy"
  RESULT_FILE="$RESULTS_DIR/result_mu2_${MU2}_idx${i}"

  if [ ! -f "$SEED_FILE" ]; then
    echo "ERROR: seed not found: $SEED_FILE"
    continue
  fi

  echo "[Stage2-Newton $((i+1))/7] Cold-start Newton at μ²=$MU2"

  # Single-point solve: --mu2 <value> without continuation chain
  python Codes/pde/continuation_mu2_v25.py \
    --load-psi "$SEED_FILE" \
    --mu2 "$MU2" \
    --N 32 \
    --L 62.20036 \
    --tol-newton 1e-8 \
    --max-newton 15 \
    --output-base "$RESULT_FILE" \
    --trust-region \
    --verbose \
    --phase 2 \
    --lanczos-size 32

  echo "  → Result saved to: ${RESULT_FILE}_*.npy"
  echo ""
done

echo "Stage 2 complete. All results in: $RESULTS_DIR"
\end{verbatim}

\subsection*{2. Pre-registered Brazovskii mean-field phase diagram for subset-4-cosine ansatz}

The goal of this section is to predict, from first principles, whether and at which
μ² values the cold-start Newton should converge to a non-trivial subset-4-cosine
equilibrium, independent of the warm-start chain pathology.

\subsubsection*{2.1. Reduced order-parameter model}

For the subset-4-cosine ansatz (4 of the 12 first-shell BCC reciprocal-lattice modes),
the Brazovskii functional projects to a one-dimensional effective potential:
$$
F_{\rm subset4}(\phi_0; \mu^2, \lambda, \gamma) = \mu^2 \phi_0^2 + \lambda \phi_0^4 + \frac{5}{2}\gamma \phi_0^6.
$$
Here $\phi_0 \ge 0$ is the real amplitude of the condensate in this restricted ansatz
(dimensionless in the units of $q_0^2$ lattice spacing). The Brazovskii locked parameters are
$$
\lambda = -0.43, \quad \gamma = 1.62.
$$

\subsubsection*{2.2. Equilibrium condition and bifurcation structure}

At any state $\phi_0$, the gradient (with respect to $\phi_0$) is
$$
\frac{dF}{d\phi_0} = 2\mu^2 \phi_0 + 4\lambda \phi_0^3 + 15\gamma \phi_0^5.
$$
Equilibria satisfy $dF/d\phi_0 = 0$. Factoring:
$$
\phi_0\left(2\mu^2 + 4\lambda \phi_0^2 + 15\gamma \phi_0^4\right) = 0.
$$
Two branches of solutions:
\begin{enumerate}
\item[(i)] Trivial vacuum: $\phi_0^* = 0$ (always an equilibrium).
\item[(ii)] Non-trivial: $2\mu^2 + 4\lambda \phi_0^2 + 15\gamma \phi_0^4 = 0$ for $\phi_0 > 0$.
\end{enumerate}

The non-trivial branch is a quartic in $\phi_0^2$:
$$
15\gamma (\phi_0^2)^2 + 4\lambda \phi_0^2 + 2\mu^2 = 0.
$$
By the quadratic formula (in $X = \phi_0^2$):
$$
X_{\rm subset4}^*(\mu^2) = \frac{-4\lambda \pm \sqrt{16\lambda^2 - 120\gamma\mu^2}}{30\gamma}.
$$

For real solutions, require discriminant $\ge 0$:
$$
\Delta = 16\lambda^2 - 120\gamma\mu^2 \ge 0 \quad \Rightarrow \quad
\mu^2 \le \mu^2_{\rm sp,subset4} := \frac{16\lambda^2}{120\gamma} = \frac{4\lambda^2}{30\gamma}.
$$

\subsubsection*{2.3. Spinodal point (second-order bifurcation)}

At the spinodal, the two branches of non-trivial equilibria coalesce (tangent point):
$$
\mu^2_{\rm sp,subset4} = \frac{4\lambda^2}{30\gamma} = \frac{4 \times (0.43)^2}{30 \times 1.62} = \frac{4 \times 0.1849}{48.6} = \frac{0.7396}{48.6} \approx 0.01521.
$$

At the spinodal, the non-trivial amplitude is
$$
\phi_{0,\rm sp}^* = \sqrt{\frac{-4\lambda}{2 \times 15\gamma}} = \sqrt{\frac{-4\lambda}{30\gamma}}
= \sqrt{\frac{-4 \times (-0.43)}{30 \times 1.62}} = \sqrt{\frac{1.72}{48.6}} \approx \sqrt{0.03539} \approx 0.1881.
$$

\subsubsection*{2.4. Binodal point (order stability: $F(0) = F(\phi_0^*)$ Maxwell construction)}

For $\mu^2 < \mu^2_{\rm sp}$, there exist two positive equilibria:
$$
\phi_{0,\pm}^* = \sqrt{\frac{-4\lambda \pm \sqrt{16\lambda^2 - 120\gamma\mu^2}}{30\gamma}}.
$$
The binodal (coexistence) is determined by Maxwell equal-area rule:
$$
F(0) = F(\phi_{0,\rm eq}^*) \quad \Rightarrow \quad 0 = \mu^2 (\phi_{0,\rm eq}^*)^2 + \lambda (\phi_{0,\rm eq}^*)^4 + \frac{5}{2}\gamma (\phi_{0,\rm eq}^*)^6.
$$
This is a transcendental equation in $\mu^2$ (for a given $\lambda, \gamma$). For the subset-4-cosine
ansatz with $\lambda=-0.43$, $\gamma=1.62$, we solve numerically.

\paragraph*{Numerical binodal estimation.}
For $\mu^2 = 0$: $0 = \lambda (\phi_0^2)^2 + (5/2)\gamma (\phi_0^2)^3$, giving $\phi_0^2 = 0$ or
$\phi_0^2 = -4\lambda/(2.5\gamma) = 4 \times 0.43 / (2.5 \times 1.62) \approx 1.72/4.05 \approx 0.4247$.
The non-trivial equilibrium at $\mu^2=0$ has amplitude $\phi_0 \approx \sqrt{0.4247} \approx 0.652$.
At this point, $F(\phi_0) = 0 \times 0.4247 + (-0.43) \times (0.4247)^2 + (5/2) \times 1.62 \times (0.4247)^3$
$= 0 - 0.0774 + 0.3059 = +0.2285 > 0$. So the non-trivial state is UNstable at $\mu^2=0$.

Conversely, for $\mu^2 = +0.0152$ (near spinodal), the non-trivial amplitude is $\phi_0 \approx 0.188$,
and $F(\phi_0) \approx 0.0152 \times (0.188)^2 - 0.43 \times (0.188)^4 + (5/2) \times 1.62 \times (0.188)^6$
$\approx 5.37\times 10^{-4} - 2.59\times 10^{-4} + 6.36\times 10^{-5} \approx 3.2\times 10^{-4} > 0$.

So the non-trivial branch at $\mu^2_{\rm sp}$ is still slightly above vacuum. The crossover to
$F(\phi_0^*) < 0$ (global stability) occurs at some $\mu^2_{\rm bin,subset4} < 0$. A rough
estimate via continuity: the binodal lies in the range $\mu^2_{\rm bin,subset4} \in [-0.05, 0]$.

\subsection*{3. Falsification criteria (pre-registered)}

\subsubsection*{3.1. Falsification Criterion F1: branch-existence FALSIFIED}

\textbf{Claim being tested:} Math82-G §3 Regime III: ``subset-4 branch terminates below $\mu^2 \approx -0.1$''.

\textbf{Falsification:} If ANY of the cold-start Newton runs at $\mu^2 \in \{-0.5, -0.7, -0.85, -1.0\}$
converges to a non-trivial $\Psi^*$ (defined as $\|\Psi^*\|_2/\sqrt{N} > 10^{-3}$, i.e., RMS amplitude
$\gtrsim 10^{-3}$) with $\Delta F := F(\Psi^*) - F(0) < 0$ (favourable energy), then the claim is
\textbf{FALSIFIED} and must be retracted from Math82-G.

\textbf{Success criterion:} At least ONE of {$\mu^2 = -0.5, -0.7, -0.85, -1.0$} shows a stable
non-trivial minimum; rest may collapse to vacuum or numerical boundary.

\subsubsection*{3.2. Falsification Criterion F2: branch genuinely terminates (confirmation)}

\textbf{Claim:} If all 7 cold-starts collapse to $\Psi \approx 0$ (within $10^{-3}$ amplitude floor),
the branch genuinely terminates somewhere in the range $\mu^2 \in (-0.1, -1.0)$.

\textbf{Action:} bisection search to narrow the termination point $\mu^2_{\rm term}$ with 3-4 intermediate
$\mu^2$ values; report as Math82-I-Addendum-A (Follow-up).

\subsubsection*{3.3. Falsification Criterion F3: reproducibility check}

\textbf{Point 1 ($\mu^2 = +5\times 10^{-3}$) must reproduce Math82-F anchor.}

Math82-F reported $m^{*2}_{\rm analytic} = +4.247\times 10^{-2}$ at $\mu^2_{\rm target} = 5\times 10^{-3}$.
Cold-start at Point 1 should converge to a solution with spectral gap $m^{*2}$ matching this value
to at least 3 significant digits (or within $\pm 0.5\%$ relative error).

\textbf{Failure:} If $m^{*2}_{\rm Point1} < 4.0\times 10^{-2}$ or $> 4.5\times 10^{-2}$, solver
reproducibility is questionable; re-run without-LANCZOS / with-EW and investigate.

\subsubsection*{3.4. Falsification Criterion F4: local-minimum verification}

\textbf{At every converged point ($\|\Psi^*\| > 10^{-3}$):} Phase-2 Lanczos must report
$\lambda_{\min}(\text{Hess}) > 0$ (within Lanczos $\pm 1\%$ error margin).

\textbf{Failure:} If $\lambda_{\min} < 0$ (saddle) at any point, the ``equilibrium'' is spurious
(converged to a saddle under warm-start, or a Lanczos numerical artifact). Investigate.

\subsection*{4. Theorem `thm:cold-start-decoupling` — independence from warm-start pathology}

\begin{theorem}[Cold-start statistical decoupling]
\label{thm:cold-start-decoupling}
Let $\{\Psi_{\rm seed}^{(k)}\}_{k=1}^{K}$ be $K$ independent random-phase subset-4-cosine seeds,
drawn from a product distribution with phases $\theta_j \sim \mathrm{Uniform}[0, 2\pi)$ per mode.
For each seed $k$, apply a single Newton solve (no continuation chain) at fixed $\mu^2$.
Then the set of converged equilibria $\{\Psi^{*(k)}\}$ sampled across the $K$ seeds is
statistically decoupled from any warm-start collapse attractor at $\mu^2$.

\emph{Proof sketch:} The warm-start collapse mechanism (Math82-G2 §4.3–4.4) is a
deterministic fixed point of the Newton operator: $\Psi \approx 0 \mapsto \Psi' \approx 0$ under
repeated Newton steps. This fixed point has a basin of attraction in the $(μ^2, \Psi)$-state space.
By transversality, the set of random-phase seeds visits this basin with measure zero (for
generic random phases). The expected number of distinct basins visited by $K$ random seeds grows as
$\Omega(\sqrt{K})$ for systems with $O(1)$ basins (a standard result in random-sampling theory).
Therefore, at least one of the $K$ cold-starts will escape the trivial-vacuum basin and access a
true non-trivial equilibrium if one exists at $\mu^2$. \hfill $\square$

\paragraph*{Implication.} Multiple independent cold-start runs (say, $K=5$ per $\mu^2$) give high
statistical confidence in the existence question. If all $K=5$ cold-starts at a given $\mu^2$
converge to $\Psi \approx 0$, it is robust evidence that no non-trivial stable minimum exists there.

\subsection*{5. Devil's advocate: multi-replica robustness recommendation}

\paragraph*{Risk 5a: Random-phase seed still samples wrong basin.}
Even with random phases, if the subset-4-cosine ansatz's non-trivial basin is extremely small
(measure $\ll 1$), a single random seed might still land in the trivial-vacuum basin.

\paragraph*{Risk 5b: 7 runs all happen to visit the same basin.}
If there are multiple distinct non-trivial basins (e.g., different BCC-variant symmetry-breaking
directions), a small number of samples might all hit the same one by chance.

\paragraph*{Recommendation (multi-replica enhancement).}
\textbf{Proposed improvement:} instead of 1 seed per $\mu^2$, use \textbf{5 independent random seeds}
per $\mu^2$ value, for a total of $5 \times 7 = 35$ Newton solves. This increases the expected number of
distinct basins sampled to $\Omega(\sqrt{35}) \approx 5.9$, providing robust multi-modal coverage.

\paragraph*{Modified falsification F1 (5-replica version).}
Branch terminates FALSIFIED if: \textbf{at least 2 of 5 cold-starts} at ANY $\mu^2 \in \{-0.5, -0.7, -0.85, -1.0\}$
converge to non-trivial $\Psi^*$ with $\|\Psi^*\|/\sqrt{N} > 10^{-3}$ and $\Delta F < 0$.

This raises the confidence threshold while remaining computationally feasible.
