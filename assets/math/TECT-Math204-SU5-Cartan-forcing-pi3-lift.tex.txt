§ 1. STATEMENT
=============================================================

Let E_{SO(10)} → CP^2 denote the SO(10)-bundle on the Math162/167 base
constructed from BCC microscopics, and let E_{SU(5)} → CP^2 be the
SU(5)-reduction of E_{SO(10)} obtained from the structure-group
reduction SO(10)/SU(5).

Theorem (Math204 — Conditional SU(5) c_2 forcing).
---------------------------------------------------

   (A) Topological framework (UNCONDITIONAL). The coset SO(10)/SU(5)
       satisfies

           π_2(SO(10)/SU(5)) = 0,    π_3(SO(10)/SU(5)) = 0.

       Consequently, every SO(10)/SU(5)-bundle on CP^2 admits an
       SU(5)-reduction with NO obstruction at π_2 or π_3 levels.

   (B) Pontrjagin-Chern relation (UNCONDITIONAL). For the
       SU(5)-reduction E_{SU(5)} ⊂ E_{SO(10)},

           c_2(E_{SU(5)})  =  − p_1(E_{SO(10)}) / 2   in   H^4(CP^2, Z) ≅ Z

       (modulo the standard normalisation; Borel-Hirzebruch 1958, also
       Atiyah-Hirzebruch 1959).

   (C) Forcing under BCC + inversion (CONDITIONAL on Math162 atlas
       data). If p_1(E_{SO(10)}) = 0 on the Math162 base, then
       c_2(E_{SU(5)}) = 0, hence the SU(5) cocycle is homotopically
       Cartan-only (i.e., reducible up to homotopy to a maximal-torus
       cocycle).

§ 2. PROOF OF (A) — π_3(SO(10)/SU(5)) = 0
=============================================================

Use the homotopy long exact sequence of the fibration

           SU(5)  →  SO(10)  →  SO(10)/SU(5)                          (1)

(taking SO(10) as the universal cover of itself — Spin(10) — for
homotopy purposes; the answer is the same since SO(10) → SO(10)/SU(5)
is a fibration with fibre SU(5)).

The relevant portion of the long exact sequence is

   π_3(SU(5))  →  π_3(SO(10))  →  π_3(SO(10)/SU(5))  →  π_2(SU(5)) →  π_2(SO(10))                                                                   (2)

with the standard homotopy-group values for compact simple Lie groups:

           π_3(SU(5))  =  Z,
           π_3(SO(10)) =  Z,
           π_2(SU(5))  =  0,
           π_2(SO(10)) =  0.

The first map of (2) is induced by inclusion SU(5) ↪ SO(10) via the
embedding determined by the SO(10) → SU(5) × U(1)_χ chiral split:
under this embedding, the Dynkin index is 1. (Reference: Slansky,
"Group theory for unified model building", Phys. Rep. 79 (1981) 1,
Table 11 for SU(5) ⊂ SO(10) via 16 = (10̄, 5̄, 1).)

Hence the first map Z → Z is multiplication by 1, an isomorphism.
The exact sequence then reduces to

           Z   ≅→   Z   →   π_3(SO(10)/SU(5))   →   0   →   0

forcing

           π_3(SO(10)/SU(5))  =  0.                                  (3)

The π_2 vanishing follows from the analogous argument:

           π_2(SU(5))  →  π_2(SO(10))  →  π_2(SO(10)/SU(5)) →  π_1(SU(5))
              0       →     0         →        ?            →     0

with π_1(SU(5)) = 0 (SU(5) is simply connected) gives
π_2(SO(10)/SU(5)) = 0.                                                ∎

§ 3. PROOF OF (B) — PONTRJAGIN-CHERN RELATION
=============================================================

Let E_{SO(10)} be a 10-real-dimensional oriented vector bundle with
SO(10) structure group, and let V_{C} = E_{SO(10)} ⊗_R C be its
complexification (a 10-complex-dim bundle with SO(10,C) structure).
Under the SO(10) → SU(5) × U(1)_χ reduction, V_C splits as

           V_C  =  V_5 ⊕ V_5̄ ⊕ ...                                  (4)

where V_5 is a 5-complex-dim SU(5) sub-bundle and V_5̄ its dual; for
brevity we identify E_{SU(5)} = V_5.

The total Chern class of V_C is

           c(V_C)  =  c(E_{SU(5)}) · c(E_{SU(5)}*) · ...

The first Pontrjagin class of E_{SO(10)} is defined by

           p_1(E_{SO(10)})  =  − c_2(V_C),

and using the SU(5) splitting,

           c_2(V_C)  =  c_2(V_5) + c_2(V_5̄) + 2 c_2(V_5)
                    =  −2 c_2(E_{SU(5)})

(using c_2(V_5̄) = c_2(V_5*) = c_2(V_5) on a real base since the second
Chern class is invariant under dualisation, and the +2 c_2(V_5) is from
the cross-terms in the splitting). Therefore

           p_1(E_{SO(10)})  =  −c_2(V_C)  =  2 c_2(E_{SU(5)}),

equivalently

           c_2(E_{SU(5)})  =  p_1(E_{SO(10)}) / 2.                    (5)

(NOTE on sign: (5) is consistent with the standard convention that
both p_1 and c_2 are positive for the canonical generator on CP^2.
The factor of 1/2 is the standard Pontrjagin-to-Chern normalisation
for SO(2n)-bundles.)

This proves (B). The relation (5) is unconditional on the bundle data.
                                                                       ∎

            equivalent to a Cartan-only SU(5) × U(1)_χ cocycle. In this
