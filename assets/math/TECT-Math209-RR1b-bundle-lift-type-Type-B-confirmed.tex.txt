§ 2. PROOF — DIRECT FROM MATH162'S EXPLICIT TRANSITION
=============================================================

  Step 1.  Restriction to the Cartan U(1) subbundle.

  The Math162 transition g_{01} = exp(i·arg(u_1)·T_3) takes values in
  the maximal torus T = U(1)^4 ⊂ SU(5), specifically in the U(1)
  generated by T_3. Restricting to this U(1)-subbundle reduces (1)
  to a U(1)-cocycle

       h_{01}(u_1, u_2)  =  exp(i · arg(u_1)).                      (3)

  This is the line-bundle transition function we test for
  σ_I-equivariance.

  Step 2.  σ_I-action on the homogeneous coordinates.

  By Math207 §2 Step 6 (RR1a closure, ACCEPT), σ_I acts on CP²
  homogeneous coordinates as

       σ_I : [u_0 : u_1 : u_2]  ↦  [ū_0 : ū_1 : ū_2].              (4)

  Step 3.  σ_I-action on the transition function.

  Substituting (4) into (3):

       (σ_I^* h_{01})(u_1, u_2)  =  h_{01}(σ_I(u_1, u_2))
                                  =  h_{01}(ū_1, ū_2)
                                  =  exp(i · arg(ū_1))
                                  =  exp(−i · arg(u_1))
                                  =  h_{01}(u_1, u_2)^{−1}.          (5)

  In the second-to-last step we used arg(ū_1) = −arg(u_1) (a basic
  fact about complex conjugation acting on the principal argument).
  In the last step we used the U(1) inversion h^{−1} = e^{−i·arg(u_1)}.

  Step 4.  Identification of the equivariance type.

  For an abelian U(1) line bundle, the inverse of a transition
  function equals its complex conjugate:

       h^{−1}  =  e^{−i·arg(u_1)}  =  conj(e^{i·arg(u_1)})  =  h̄.   (6)

  Therefore (5) reads

       σ_I^* h_{01}  =  h̄_{01}.                                    (7)

  This is precisely the Type B (Real / anti-linear) equivariance:
  the σ_I-pulled-back transition equals the COMPLEX CONJUGATE
  transition, NOT the identical transition. By the standard
  classification of bundle equivariance, (7) corresponds to

       L  ≅  σ_I^* L̄                                                 (8)

  (NOT L ≅ σ_I^* L, which would be Type A).

  Step 5.  Consequence: c_1(L) is UNCONSTRAINED by σ_I-equivariance.

  Under (8), Math203's argument fails:

       c_1(L)  =  c_1(σ_I^* L̄)
              =  σ_I^*(c_1(L̄))
              =  σ_I^*(−c_1(L))                  [conjugation flips sign of c_1]
              =  −σ_I^*(c_1(L)).

  Using σ_I^* h = −h on H^2(CP², ℤ) (Atiyah's Lemma 2.1, retained
  from Math203):

       c_1(L)  =  −(−c_1(L))  =  c_1(L),                            (9)

  which is a TAUTOLOGY — true for any c_1(L) = k·h, k ∈ ℤ. Type B
  equivariance imposes NO constraint on the first Chern class.

  Step 6.  Verification with Math162's own c_1 = 1.

  Math162 §4.2 computes c_1(𝒫) = 1 from the explicit transition (1).
  This is a non-trivial integer, INCONSISTENT with the Type A
  conclusion c_1 = 0 but FULLY CONSISTENT with Type B (which leaves
  c_1 unconstrained). The internal consistency of Math162 (between
  its explicit transition function and its computed c_1 = 1)
  confirms that Type B is the correct equivariance type.       ∎
