\subsection{Berry Connection on Moduli Space}

The Berry phase is a geometric phase arising from parallel transport of the ground state along a loop in parameter space. The corresponding connection form is:
\begin{equation}
\mathcal{A}_{\mathrm{Berry}} = i \langle \Psi | \nabla_\mu \Psi \rangle \, d x^\mu,
\end{equation}
where $|\Psi\rangle$ is a continuously chosen ground-state representative on each patch.

For the defect moduli space $M_{\mathrm{BCC}}$, the ground state varies with the position $p \in M_{\mathrm{BCC}}$. The corresponding Berry connection is a connection on the line bundle $\mathscr{L} \to M_{\mathrm{BCC}}$.

**Claim 2.2** (Topological triviality): The Berry line bundle $\mathscr{L} \to M_{\mathrm{BCC}}$ is trivial on the three-patch cover because $M_{\mathrm{BCC}}$ is simply connected.

**Proof of Claim 2.2**:
\begin{enumerate}[label=(\roman*)]
\item A line bundle over a simply-connected space is determined by its first Chern class $c_1(\mathscr{L}) \in H^2(M_{\mathrm{BCC}}, \mathbb{Z})$.
\item For a contractible space, $H^2(M_{\mathrm{BCC}}) = 0$.
\item Therefore $c_1(\mathscr{L}) = 0$, so $\mathscr{L}$ is trivial: $\mathscr{L} \cong M_{\mathrm{BCC}} \times \mathbb{C}$.
\item A trivial line bundle admits a global holomorphic section (e.g., the constant section $\sigma(p) = 1$ for all $p$).
\end{enumerate}

\subsection{Consequence: Vanishing of the Cocycle}

If $\mathscr{L}$ is trivial, then there exists a global, single-valued ground-state representative $|\Psi_{\mathrm{global}}(p)\rangle$ that varies continuously over all of $M_{\mathrm{BCC}}$.

Along the transition between patches, the Berry phase is:
\begin{equation}
\beta_{ij} = \int_{\gamma_{ij}} \mathcal{A}_{\mathrm{Berry}},
\end{equation}
where $\gamma_{ij}$ is the transition curve.

On a triple overlap $U_0 \cap U_1 \cap U_2$, the cocycle is:
\begin{equation}
\beta_{012} = \beta_{01} + \beta_{12} + \beta_{20}.
\end{equation}

**Theorem 2.3** (Berry Cocycle Vanishing):
\begin{equation}
\beta_{012} \equiv 0 \pmod{2\pi}.
\end{equation}

**Proof of Theorem 2.3**:
\begin{enumerate}[label=(\alph*)]
\item The global ground-state section $|\Psi_{\mathrm{global}}(p)\rangle$ is single-valued on $M_{\mathrm{BCC}}$.
\item On each patch $U_i$, we may express $|\Psi_{\mathrm{global}}\rangle$ in local coordinates and compute the Berry phase.
\item Along a closed curve in $M_{\mathrm{BCC}}$ (including the boundary of triple overlaps), the Berry phase returns to zero:
\begin{equation}

\item Decomposing the boundary into the three transition edges:
\begin{equation}
\int_{\gamma_{01}} + \int_{\gamma_{12}} + \int_{\gamma_{20}} = 0 \pmod{2\pi}.
\end{equation}
\item Therefore: $\beta_{012} = 0 \pmod{2\pi}$.  \hspace{3cm} $\square$
\end{enumerate}
