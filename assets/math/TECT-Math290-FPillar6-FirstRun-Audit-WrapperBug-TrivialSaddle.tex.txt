\section{Run identification and headline result}

We report on the first production execution of the Math236 \S9.2
four-stage continuum-limit-scan protocol against the F-Pillar6
falsification gate (Math286 \S5). The run terminated after
$\Delta t_{\rm wall} = 39394.1\,\mathrm{s} \approx 10.94\,\mathrm{h}$
on a single $N$-point ($N=16$) with the Math236 wrapper reporting
\verb|OVERALL: PARTIAL| and all six falsification gates G1--G6
returning \verb|FAIL|. The wrapper interpreted the run as a
\emph{G3 sign-consistency} violation (extracted amplitude
$f \le 0$) and triggered fail-fast, skipping $N \in \{32, 64, 128\}$.

The headline finding of the present audit is that this verdict is
\emph{not} a physical falsification of the F-Pillar6 gate: the run
output exposed a triad of concurrent defects, two in the Math236
wrapper and one in the seed-induced Newton trajectory. The
falsification-gate calendar entry F-Pillar6 (Math286 \S5,
deadline 2026-05-29) is therefore \emph{not consumed}.

\section{Empirical re-extraction with patched code}

We applied the corrected amplitude extractor (\S3 below) to the
saved \verb|Psi_final.npy| produced by the original run, holding
$\mu^{2}, L_{\rm box}, q_{0}$ and the discrete grid fixed:

\begin{equation}
f^{\rm patched}(a=1.0) \;=\; 2.591384 \times 10^{-5},
\qquad
\text{status} = \texttt{OK}, \quad
\text{shell modes} = 8, \quad
n_{\rm comp} = 3.
\label{eq:fpatch}
\end{equation}

The reported amplitude is $f \sim 10^{-5}$, four orders of magnitude
below the broken-phase expectation $f \sim O(1)$ at $\mu^{2}=-0.7$.
Equation~\eqref{eq:fpatch} simultaneously falsifies the two wrapper
bugs (A and B, \S3--\S4) \emph{and} confirms the third defect: the
Newton-Krylov solver converged to a near-trivial configuration
($\|\Psi^\star\|_{L^{2}} \sim 10^{-2}$) rather than to the
broken-phase minimum, regardless of any extraction issue.

\section{Bug A --- amplitude extractor cannot handle vector fields}

The v25 driver persists the order parameter as
\verb|Psi_final.npy| with \verb|shape=(3,16,16,16)|, \verb|dtype=complex128|
(three-component compression / vector mode). The Math236 wrapper's
\verb|extract_amplitude| (originally lines 329--356 of
\verb|Math236_continuum_limit_scan.py|) implemented the BCC primitive
shell projector
\begin{equation}
f \;=\;
\frac{1}{N_{\rm shell}}
\Bigl(
\sum_{|\mathbf{k}|\,\approx\,q_{0}}|\Psi_{\mathbf{k}}|^{2}
\Bigr)^{1/2},
\quad N_{\rm shell} = 12,
\label{eq:fdef-scalar}
\end{equation}
under the explicit assumption \verb|arr.ndim == 3|, returning $0.0$
for any vector field via the guard
\verb|if arr.ndim != 3: return 0.0, "unexpected ndim=4..."|.

\paragraph{Patch.} Equation~\eqref{eq:fdef-scalar} is replaced by the
multi-component shell norm
\begin{equation}
f \;=\;
\frac{1}{N_{\rm shell}}
\Bigl(
\sum_{c=1}^{n_{\rm comp}}
\sum_{|\mathbf{k}|\,\approx\,q_{0}}
|\Psi_{\mathbf{k},c}|^{2}
\Bigr)^{1/2},
\label{eq:fdef-vector}
\end{equation}
which reduces to~\eqref{eq:fdef-scalar} when $n_{\rm comp}=1$ and is
the natural $L^{2}(S^{q_{0}}; \mathbb{C}^{n_{\rm comp}})$ shell norm
for an $n_{\rm comp}$-component complex field. The patched extractor
also returns an explicit status string
\verb|status \in {OK, BAD_NDIM, BAD_SHAPE, NO_SHELL_MODES, READ_ERROR}|
so the calling site can distinguish a structural defect from a
genuine $f=0$ physics result. The patched implementation is
empirically validated by \eqref{eq:fpatch}.

\section{Bug B --- extraction status flag overwrites the failure note}

The original wrapper unconditionally set
\verb|per.extraction_status = "OK"| immediately after every call to
\verb|extract_amplitude|, regardless of whether the returned
amplitude was genuine or a structural-failure stub:

\begin{verbatim}
f_val, note = extract_amplitude(psi, q0=q0, N=N, L_box=L_box)
per.f_amplitude = f_val
per.extraction_status = "OK"             # <-- bug
per.extraction_notes = note
\end{verbatim}

The fail-fast gate (\verb|per_point_fatal_reason|) trusted the
\verb|"OK"| label and only triggered on $f \le 0$ via the G3
sign-consistency predicate. The result was that a code-bug zero was
misclassified as a physical sign-consistency violation, terminating
the scan after the first $N$-point.

\paragraph{Patch.} The extractor returns \verb|(f, note, status)| and
the caller propagates \verb|status| faithfully:

\begin{verbatim}
f_val, note, status = extract_amplitude(psi, q0=q0, N=N, L_box=L_box)
per.f_amplitude = f_val
per.extraction_status = status           # OK / BAD_NDIM / ...
per.extraction_notes  = note
\end{verbatim}

\verb|per_point_fatal_reason| now distinguishes
\verb|extraction_structural_failure| (a code defect) from the G3
sign-consistency violation (a genuine physics result). Both are
fail-fast triggers, but the messages and the post-mortem actions
differ: a structural failure requires a code patch; a G3 fail
requires a physics rethink.

\section{Bug C --- trivial-saddle convergence and the Stage-1.5 free-energy guard}

The third defect is structural to the Newton trajectory, not the
wrapper. The solver self-reports \verb|converged=True| at
$\tau = 1\times 10^{-5}$, but the converged $\Psi^\star$ has
$F(\Psi^\star) = +3.75\times 10^{-6} > F(0) = 0$ and a Hessian with
$\lambda_{\min} = -4.78\times 10^{-1} < 0$. By the standard
Brazovskii diagnosis, $\Psi^\star$ is a \emph{near-trivial saddle}
(unstable critical point with $\Psi^\star \approx 0$,
$\|\Psi^\star\|_{L^{2}} \sim 10^{-2}$), not the broken-phase
minimum that the F-Pillar6 protocol is designed to extract from.

The most plausible cause is the combination of
(i) a coarse lattice $N=16$ for which $q_{0}/\Delta k \approx 1.73$
is a fractional mode index, with no exact $q_{0}$ representable on
the discrete reciprocal lattice;
(ii) a default symmetric/random seed with no preferred direction
$\hat{\mathbf{n}} \in S^{2}$; and
(iii) a slack residual tolerance $\tau = 10^{-5}$ that tightens only
the residual norm, not the basin-of-attraction selectivity.
Together these allow the Newton trajectory to settle on the
near-trivial saddle rather than locating the broken-phase minimum.
The 10.94-hour wall time is consistent with Newton-Krylov
struggling near an indefinite Hessian region (slow GMRES convergence
when the operator has a near-zero eigenvalue mixing with the
saddle's unstable direction).

\paragraph{Patch (Stage-1.5 free-energy guard).} A new helper
\verb|parse_delta_F(stdout_tail)| extracts $\Delta F$ and the
``favorable'' verdict from the solver stdout and stamps them into
\verb|PerNResult.delta_F| / \verb|PerNResult.favorable|. The
fail-fast gate then returns
\begin{equation}
\text{TRIVIAL\_SADDLE}
\;\;\text{iff}\;\;
f > 0 \;\wedge\; \Delta F \ge 0,
\label{eq:saddle-gate}
\end{equation}
which is fatal (so fail-fast halts the scan) but distinguishable
from G3. The operator response is to re-seed via the new helper
\verb|Codes/supplementary/Math236_seed_striped.py|, which generates
a deterministic striped initial configuration
\begin{equation}
\Psi^{(0)}_{c}(\mathbf{x})
=
A_{0}\,\hat{e}^{\,c}\,\cos(q_{0}\,\hat{\mathbf{n}}\cdot\mathbf{x}),
\qquad A_{0} = 0.5,
\;\; \hat{e} = (1,0,0),
\;\; \hat{\mathbf{n}} = (1,0,0),
\label{eq:seed}
\end{equation}
and saves it as a \verb|.npy| file passable to the v25 driver via
\verb|--load-psi|.

\section{Devil's-advocate self-test (CLAUDE.md \S6.3.1)}

\paragraph{$\alpha$.} ``The $f \sim 10^{-5}$ extracted by the patched
code is just lattice noise; the run is genuinely $f \to 0$ and
F-Pillar6 has fired in the wrong direction.''
\textbf{DISMISSED.} The supporting evidence for ``trivial saddle''
is not the small $f$ alone -- it is the simultaneous
$\Delta F = +3.75\times 10^{-6} > 0$ (energy above trivial vacuum)
and $\lambda_{\min} = -0.48 < 0$ (indefinite Hessian). A genuine
broken-phase minimum has $\Delta F < 0$ and a positive-semidefinite
Hessian on the transverse modes. The triad
$(\Delta F > 0,\; \lambda_{\min} < 0,\; f \ll 1)$ is canonical
near-trivial-saddle phenomenology.

\paragraph{$\beta$.} ``Maybe $\mu^{2} = -0.7$ is on the symmetric
side of the lattice-dressed Brazovskii transition at $N=16$ and
the broken phase genuinely does not exist on this grid.''
\textbf{VALID-WITH-MITIGATION.} Coarse-lattice mode quantization
shifts the effective transition; at $N=16$, $L=16$ the discrete
reciprocal lattice has $\Delta k = 2\pi/16 \approx 0.393$ and the
target $q_{0} = 0.68$ falls between the $|n|=1$ and $|n|=2$ shells
(modes at $k = 0.393$ and $k = 0.785$). It is conceivable that
$\mu^{2} = -0.7$ is genuinely above the dressed transition for
$N=16$. The mitigation is exactly the planned re-run: if all of
$N \in \{16,32,64\}$ converge to trivial saddles even with the
striped seed, that is itself diagnostic information about where
the Brazovskii transition sits on the discrete lattice -- it
tells us we need either smaller $\mu^{2}$ (e.g.~$-1.0$) or finer
$N$ to enter the broken phase.

\paragraph{$\gamma$.} ``The 10.94-hour wall time per $N$-point is
unacceptable; even with the patches, the full $N$-list of
$\{16,32,64,128\}$ would cost months.''
\textbf{UPHELD-WITH-CONTINGENCY.} A trivial-saddle Newton
trajectory is precisely the Hessian-indefinite regime where GMRES
convergence is worst (one near-zero eigenvalue mixing with the
saddle's unstable direction). With a striped seed in the
broken-phase basin, Newton steps land in a positive-definite
Hessian regime where GMRES is well-conditioned; the wall time
should drop by an order of magnitude. If the re-run confirms this,
follow-up task: revisit the Newton tolerance at $\tau = 10^{-6}$
once the basin is correct.

\section{Quantitative sanity checks (CLAUDE.md \S6.3.4)}

\begin{itemize}
\item \textbf{Dimensional}: $[f] = [\Psi]$ via
Equation~\eqref{eq:fdef-vector}; $[\Psi^\star]$ matches
\verb|complex128| persisted by v25; PASS.
\item \textbf{Magnitude}: $\|\Psi^\star\|_{L^{2}}^{2} \approx 3.79\times 10^{-4}$,
$F = (\mu^{2}/2)\|\Psi\|^{2} + (\lambda/4)\|\Psi\|^{4} + \cdots$
gives $F \sim (-0.7/2)(3.8\times 10^{-4}) + \cdots \sim -1.3\times 10^{-4}$
to leading quadratic order. The reported $F = +3.75\times 10^{-6}$
is much smaller in magnitude than the naive quadratic estimate, which
is consistent with $\Psi^\star$ being a saddle (not the
quadratic-term-dominated minimum); PASS-with-caveat (the saddle
sits in the regime where quartic and gradient corrections cancel
the negative quadratic term).
\item \textbf{Limit case}: trivial seed $\Psi^{(0)}=0$ produces
$\Psi^\star = 0$, $f = 0$, $F = 0$. The reported values are all
``small but nonzero'' relative to that limit; PASS (consistent
with near-trivial saddle, not exact-trivial).
\item \textbf{Sign-direction}: $F > 0$ excludes the broken phase as a
candidate, confirming the saddle interpretation; PASS.
\item \textbf{Reproducibility}: the patched extractor is
deterministic; running it twice on the same
\verb|Psi_final.npy| produces identical output (verified);
PASS.
\end{itemize}
