\section{2. Definitions: Separable vs. Full-Branch Configuration Spaces}

\subsection{2.1 Configuration Space on $\Sigma_0$}

The gauge-field configuration space on $\Sigma_0 = \mathbb{P}^1 \times \mathbb{P}^1$ is:
$$\mathcal{A}_\Sigma = \{A_\mu(x_1, x_2) \, : \, A_\mu \in \Omega^1(\Sigma_0) \otimes \mathfrak{g}_{\rm SM}\}.$$

In coordinates:
$$\mathcal{A}_\Sigma = \{(A_{1,\mu_1}(x_1), A_{2,\mu_2}(x_2), \tilde{A}_{12}(x_1, x_2)) \,:\, \tilde{A}_{12} \in \Omega^1(\mathbb{P}^1 \times \mathbb{P}^1) \otimes \mathfrak{g}\}.$$

The "mixed-coordinates" component $\tilde{A}_{12}$ couples the two $\mathbb{P}^1$ factors.

\subsection{2.2 Separable Branch}

The separable branch is the submanifold:
$$\mathcal{A}_{\Sigma,\rm sep} = \{A_\mu \, : \, A = A_1 \otimes \mathbf{1} + \mathbf{1} \otimes A_2, \, \tilde{A}_{12} = 0\}.$$

This is a codimension-infinite submanifold (every non-separable mode is set to zero). The path integral restricted to $\mathcal{A}_{\Sigma,\rm sep}$ is:
$$\mathcal{Z}_{\rm sep} = \int_{\mathcal{A}_{\Sigma,\rm sep}} \mathcal{D}A_1 \, \mathcal{D}A_2 \, \Delta_{\rm FP}^{\Sigma_0}[A_1, A_2] \cdots$$

Math260 proves that $\Delta_{\rm FP}^{\Sigma_0}[A_1, A_2] = \Delta_{\rm FP}^{\mathbb{P}^1}[A_1] \cdot \Delta_{\rm FP}^{\mathbb{P}^1}[A_2]$ on this branch, closing the BRST constraint (H5).

\subsection{2.3 Full Branch}

The full branch includes all non-separable fluctuations:
$$\mathcal{A}_\Sigma^{\rm full} = \mathcal{A}_{\Sigma,\rm sep} + \mathcal{A}_{\Sigma, \perp},$$
where $\mathcal{A}_{\Sigma,\perp}$ is the orthogonal complement (non-separable modes).

The full path integral is:
$$\mathcal{Z}_{\rm full} = \int_{\mathcal{A}_\Sigma^{\rm full}} \mathcal{D}A \, \Delta_{\rm FP}^{\Sigma_0}[A] \cdots$$

The FP determinant $\Delta_{\rm FP}^{\Sigma_0}[A]$ must be well-defined on the ENTIRE space $\mathcal{A}_\Sigma^{\rm full}$, not just on $\mathcal{A}_{\Sigma,\rm sep}$.

For a rigorous result, we would need:
- Explicit bound $S_{\rm sep} - S_{\perp} \geq \Delta S > 0$ for non-separable fluctuations (NOT DONE).

**Verdict on Defense A**: HEURISTIC, not suitable for T6 promotion. If pursued, it would result in T4 STRONG EVIDENCE at best, with significant assumptions and computational checks required.

\subsection{3.2 Why Defense B (Continuity Extension) is Partial}

\textbf{Partial validity**: This argument is structurally sound IF:
1. The FP operator remains elliptic on the full branch (TRUE in principle, since ellipticity is topology-independent).
2. The determinant is continuous with respect to the field topology (TRUE for operators on compact manifolds).
3. The determinant does not have isolated zeros off the separable branch (NOT VERIFIED).

\textbf{Gap}: The argument does NOT address UV-finiteness off the separable branch. The separable branch has special structure (factorized Lagrangian, product metric) that ensures UV-finiteness via zeta-function regularization. Off-separable, the Lagrangian includes non-factorizing coupling terms (Math250 §2.3), and the heat-kernel expansion of $\exp[-t \mathcal{M}_{\rm FP}]$ may include additional divergent terms if the off-separable coupling breaks the regularization scheme.

**Verdict on Defense B**: PARTIAL, addressing non-vanishing but NOT UV-finiteness. Promotion to T6 would require full heat-kernel analysis on the full-branch coupling Lagrangian, which is OPEN.
