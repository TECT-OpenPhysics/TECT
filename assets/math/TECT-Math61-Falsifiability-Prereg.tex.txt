\subsubsection{SI-unit central value and uncertainty band}

To convert dimensionless anisotropy to SI units, we use the correspondence
$$\Delta c / c \sim |\Delta\eta^{\mathrm{KE}}|.$$

At the current one-loop calculation:
$$|\Delta\eta^{\mathrm{KE}}| \le 7\times 10^{-4}.$$

The light-speed anisotropy coefficient is defined in the Standard-Model Extension (SME, Kostelecký et al.)

$$\kappa^{(c)}_{\alpha\beta} = \left( g^{\mu\nu} - \eta^{\mu\nu} \right) \text{ (SME convention)}.$$

\textbf{TECT prediction}:
\begin{equation}
  P_1: \quad |\kappa^{(c)}| \in [0, 7\times 10^{-4}] \quad \text{(central: } 3.5\times 10^{-4}\text{)}.
  \label{eq:P1-spec}
\end{equation}

\textbf{Uncertainty}: This is currently a \emph{one-loop upper bound}. Higher-loop contributions and
finite-$a$ lattice corrections are not yet included. We assign a conservative one-sigma interval:
$$P_1^{\text{central}} = 3.5\times 10^{-4}, \quad \sigma(P_1) = 2\times 10^{-4} \quad
  \Rightarrow \quad P_1 \in [1.5\times 10^{-4}, 5.5\times 10^{-4}].$$

$$P_1^{\text{fail}} = |\kappa^{(c)}| > 10^{-3}.$$

\subsection{Prior and loss function}

We adopt a \textbf{uniform credence prior} on the TECT parameter space (the $a$-dial and the
continuation pathway in $(m^*, q_0)$-space). For each prediction $P_i$, we define the \textbf{loss function}:

\begin{definition}[Falsification loss]
For prediction $P_i$ with central value $c_i$, one-sigma interval $[c_i - \sigma_i, c_i + \sigma_i]$,
and failure band $B_i$, define the loss function
\[
  \mathcal{L}_i(x) = \begin{cases}
    0 & \text{if } x \in [c_i - \sigma_i, c_i + \sigma_i] \text{ (agreement)} \\
    1 & \text{if } x \in B_i \text{ (falsified)} \\
    0.5 & \text{if } x \in [\sigma_i, 2\sigma_i] \text{ (disfavoured, not falsified)}
  \end{cases}
\]
where $x$ is the experimentally measured value.
\end{definition}

\subsection{Falsification criterion (pre-registered)}

For the \emph{collective} prediction set $\mathcal{P} = \{P_1, P_2, P_3\}$:

\begin{proposition}[Falsification rule]
The theory TECT is declared \textbf{falsified} if \emph{any one} of the following holds:
\begin{enumerate}
  \item The 95\% confidence interval $[x_i - 1.96\sigma_i^{\text{exp}}, x_i + 1.96\sigma_i^{\text{exp}}]$
    of the experimentally measured value $x_i$ excludes the one-sigma TECT band $[c_i - \sigma_i, c_i + \sigma_i]$
    for prediction $i$, \emph{and} $x_i \in B_i$ (the failure band).
  \item The measured value $x_i$ lies in the failure band $B_i$ at any confidence level.
\end{enumerate}

Conversely, the theory is \textbf{not falsified} (but possibly disfavoured) if:
\begin{enumerate}
  \item $x_i$ lies within the TECT one-sigma band for at least \emph{two} of the three predictions, or
  \item $x_i$ lies in the disfavoured range (outside one-sigma but inside two-sigma) and future measurements
    with higher precision are required to fully falsify the prediction.
\end{enumerate}
\end{proposition}
