\subsection*{2. Task \#115 Evaluation (Continuum Limit)}

\subsubsection*{§2.1 Gate G1: Richardson RMS error}

\textbf{Criterion (from Math127 §4)}: For each $\mu^2$ value, the Richardson fit RMS error must satisfy:
$$\text{RMS}(m^{*2}) / \bar{m}^{*2} < 1\%.$$

\textbf{Procedure}:
\begin{enumerate}
\item For each $\mu^2 \in \{0.26, 0.22, 0.18, 0.14, 0.10\}$:
  \begin{enumerate}
  \item Fit $(h_i, m^{*2}(h_i, \mu^2))$ to $m^{*2}(h) = a_0 + a_1 h^2$ (three-point Richardson).
  \item Compute residual norm: $\text{RMS} = \sqrt{\frac{1}{3} \sum_i (m^{*2}_{\text{fit}}(h_i) - m^{*2}_{\text{data}}(h_i))^2}$.
  \item Check: RMS $/ \bar{m}^{*2} < 0.01$ ?
  \end{enumerate}
\item \textbf{Overall gate verdict}: G1 PASS if \emph{all five} $\mu^2$ values satisfy the criterion.
  G1 FAIL if \emph{any} value exceeds 1\%.
\end{enumerate}

\textbf{Action if G1 fails}: Flag Task \#115 as \emph{marginal} (1–2\% fit error). Proceed to G2 but
note that continuum limit may be unreliable. In Round T10 (synthesis), downgrade Pillar 1 status
from promoted to CONDITIONAL on reconciliation.

\subsubsection*{§2.2 Gate G2: Cross-$\mu^2$ consistency}

\textbf{Criterion (from Math127 §4)}: The continuum-limit values $m^{*2}_{\infty}(\mu^2)$ extracted from
the five Richardson fits must be mutually consistent:
$$\sigma_{\mu^2}(m^{*2}_{\infty}) < 5\% \times \overline{m^{*2}_{\infty}}.$$

\textbf{Procedure}:
\begin{enumerate}
\item Extract five continuum-limit values: $m^{*2}_{\infty}(\mu^2_j)$ for $j = 1 \ldots 5$.
\item Compute sample mean: $\overline{m^{*2}_{\infty}} = \frac{1}{5} \sum_j m^{*2}_{\infty}(\mu^2_j)$.
\item Compute sample standard deviation: $\sigma_{\mu^2} = \sqrt{\frac{1}{4} \sum_j (m^{*2}_{\infty}(\mu^2_j) - \overline{m^{*2}_{\infty}})^2}$.
\item Check: $\sigma_{\mu^2} < 0.05 \times \overline{m^{*2}_{\infty}}$ ?
\end{enumerate}

\textbf{Action if G2 fails}: Task \#115 FAILS. Diagnostic: If scatter is \emph{monotonic} (e.g., continuum
limit increases with $\mu^2$), suspect non-critical finite-size effects. If scatter is random, suspect
numerical instability in one or more runs.

\subsubsection*{§2.3 Gate G3: Slope sign}

\textbf{Criterion (from Math127 §4)}: The slope of $m^{*2}_{\infty}(\mu^2)$ vs. $\mu^2$ must be negative
(mass gap decreases toward criticality, as predicted by Brazovskii exponent $\theta_m = 1$):
$$B := \frac{dm^{*2}_{\infty}}{d\mu^2} < 0.$$

\textbf{Procedure}:
\begin{enumerate}
\item Fit the five continuum-limit points $((\mu^2 - \mu^2_c), m^{*2}_{\infty})$ to a linear model $m_\infty = A + B(\mu^2 - \mu^2_c)$.
  Use $\mu^2_c = 0.012$ (critical point from prior phase-boundary analysis).
\item Extract the slope $B$.
\item Check: $B < 0$ ?
\end{enumerate}

\textbf{Action if G3 fails}: Task \#115 FAILS. Negative slope violates Brazovskii physical expectation.

\subsubsection*{§2.4 Gate G4: Analytic match}

\textbf{Criterion (from Math127 §4)}: The continuum limit must match the \emph{analytic} prediction
from Math37 (one-loop linear-response calculation):
$$|A - 9.005| / 9.005 < 15\%.$$
where $A$ is the intercept $m^{*2}_{\infty}$ at $\mu^2 = \mu^2_c$.

\textbf{Procedure}:
\begin{enumerate}
\item From the linear fit above, extract the intercept $A = m^{*2}_{\infty}(\mu^2_c)$.
\item Compute relative deviation: $\Delta m^{*2} / 9.005 = (A - 9.005) / 9.005$.
\item Check: $|\Delta m^{*2} / 9.005| < 0.15$ ?
\end{enumerate}

\textbf{Action if G4 fails}: Task \#115 FAILS if discrepancy $> 15\%$. A 15\% discrepancy could
indicate: (a) lattice artefacts; (b) insufficient grid resolution; (c) Hartree approximation breakdown.

\subsubsection*{§2.5 Summary: Task \#115 pass/fail}

Task \#115 PASS: all four gates (G1–G4) satisfied.
Task \#115 MARGINAL: G1 fail or G2 marginal, but G3–G4 pass.
Task \#115 FAIL: any of G3–G4 fail, or G2 fails with $\sigma > 10\%$.

\subsubsection*{§3.2 Gate T2: Critical-exponent sign}

classify outcome:

\begin{enumerate}
\item \textbf{$\theta_g > 0$} (UV attractive): asymptotic safety preserved with matter. \quad \textbf{PASS}.
\item \textbf{$\theta_g = 0$ (marginal)}: boundary case, requires all-loop analysis. \quad \textbf{CONDITIONAL}.
\item \textbf{$\theta_g < 0$} (IR attractive): asymptotic non-freedom, consistent with classical field theory. \quad \textbf{EXPECTED}.
\end{enumerate}

\textbf{Procedure}:
\begin{enumerate}
\item Compute $\theta_g = 2 - 2 B_{\rm total} g_*$.
\item Classify and record result.
\end{enumerate}

\textbf{Action}: No auto-fail for T2; all three outcomes are physically meaningful. Record classification
and proceed to T3–T4.

\subsubsection*{§3.3 Gate T3: Comparison with two-loop (Task \#128)}

\textbf{Criterion (from Math133 §4)}: Once Task \#128 (2-loop $\beta_G$) is complete, verify that
two-loop corrections to matter-coupling beta-function do not destabilize the one-loop picture:
$$\Delta g_* := |g_*^{(2)} - g_*^{(1)}| / g_*^{(1)} < 30\%.$$

\textbf{Procedure}:
\begin{enumerate}
\item Compute two-loop fixed point $g_*^{(2)}$ from Task \#128 output.
\item Compute one-loop + matter fixed point $g_*^{(1,\rm matter)}$ from Task \#126.
\item Compute relative shift: $\Delta g_* = |g_*^{(2,\rm matter)} - g_*^{(1,\rm matter)}| / g_*^{(1,\rm matter)}$.
\item Check: $\Delta g_* < 0.30$ ?
\end{enumerate}

\textbf{Action if T3 fails}: Two-loop + matter effects are non-perturbative. Escalate to functional-RG.
In Round T10, note that Hartree + 2-loop is unreliable; recommend functional-RG closure for Pillar 10.

\subsubsection*{§3.4 Gate T4: Matter-hierarchy monotonicity}

and $\theta_g(N_f)$ have definite (non-flipping) dependence on $N_f$:
$$\text{sign}\left(\frac{dg_*}{dN_f}\right) = \text{const}, \quad \left| \frac{d\theta_g}{dN_f} \right| < 0.1 \text{ per species.}$$

\textbf{Procedure}:
\begin{enumerate}
\item Extract $g_*(N_f)$ for four values of $N_f$ from Task \#126 output.
\item Compute numerical derivatives $\Delta g_* / \Delta N_f$ across consecutive $N_f$ values.
\item Check: all derivatives have the same sign? Magnitude $< 0.1$ per species?
\end{enumerate}

\textbf{Action if T4 fails}: Non-monotonic behavior signals non-perturbative instability or phase transition
in the RG flow as $N_f$ varies. Flag as a sign that Hartree is unreliable.

\subsection*{4. Task \#127 Evaluation (Stress-Tensor Extraction)}

\subsubsection*{§4.1 Gate D3-α: Correlator SNR}

\textbf{Criterion (from Math128 §1)}: The two-point stress-tensor correlator $G_{\mu\nu\rho\sigma}(r)$
must have signal-to-noise ratio $> 10$ in the distance window $r \in [1, 5]$ lattice spacings:
$$\text{SNR} = \frac{|G(r)|}{|\text{noise RMS}|} > 10.$$

\textbf{Procedure}:
\begin{enumerate}
\item Parse correlator samples from Task \#127 output at distances $r = 1, 2, 3, 4, 5$.
\item Compute signal (mean value) and noise (Jackknife standard error) for each $r$.
\item Compute SNR for each $r$; require all to be $> 10$.
\end{enumerate}

\textbf{Action if D3-α fails}: Signal is dominated by noise; stress-tensor extraction is unreliable.
Recommend: increase run volume (more samples) or use a different gauge-fixing scheme (Task \#126d).

\subsubsection*{§4.2 Gate D3-β: Fit quality (low-$k$ asymptotics)}

\textbf{Criterion (from Math128 §1)}: The Fourier-transformed correlator $\widetilde{G}(\vec{k})$ must
satisfy low-$k$ asymptotic form $\widetilde{G}(k) \propto k^{-2}$ with fit RMS $< 5\%$:
$$\text{RMS}(\widetilde{G}) / |\widetilde{G}|_{\text{fit}} < 0.05.$$

\textbf{Procedure}:
\begin{enumerate}
\item Fourier-transform correlator $G(r) \to \widetilde{G}(\vec{k})$ using FFT.
\item Select low-$k$ window: $k \in [k_{\min}, k_{\max}]$ where $k_{\min} = 2\pi/L_{\text{phys}}$,
  $k_{\max} = q_0 / 3$ (well below shell scale).
\item Fit to $\widetilde{G}(k) = A_0 / k^2 + A_2 k^2 + \ldots$ (polynomial ansatz).
\item Compute fit RMS error and check criterion.
\end{enumerate}

\textbf{Action if D3-β fails}: Fit is poor, indicating contamination from intermediate-scale modes
or non-ideal correlator behavior. Recommend diagnostic: split data into three time windows and check
consistency.

\subsubsection*{§4.3 Gate D3-γ: Richardson convergence in $h$}

\textbf{Criterion (from Math128 §1)}: Repeat stress-tensor extraction at two different lattice spacings
$h_1$ and $h_2$ (e.g., $N=32$ and $N=64$ grids). The extracted $G^{\rm TECT}$ must converge:
$$|G^{\rm TECT}(h_1) - G^{\rm TECT}(h_2)| / G^{\rm TECT}(h_1) < 10\%.$$

\textbf{Procedure}:
\begin{enumerate}
\item Extract $G^{\rm TECT}$ from both grids (Method 3).
\item Compute relative difference.
\item Check: $\Delta G / G < 0.10$ ?
\end{enumerate}

\textbf{Action if D3-γ fails}: Lattice artefacts are significant. Proceed to Gate D3-δ but note that
stress-tensor result may require continuum extrapolation (additional $N=128$ run).

\subsubsection*{§4.4 Gate D3-δ: Method consistency}

\textbf{Criterion (from Math128 §1)}: Compare Method 3 (Task \#127 stress-tensor) against Methods 1
(Math45 TT extraction) and 2 (Math110-AddA Fierz-Pauli):
$$|G^{\text{Method 3}} / G^{\text{Method 1}} - 1| < 0.20, \quad |G^{\text{Method 3}} / G^{\text{Method 2}} - 1| < 0.20.$$

\textbf{Procedure}:
\begin{enumerate}
\item Retrieve $G^{\text{Method 1}}$ and $G^{\text{Method 2}}$ from pre-existing archive (Math45, Math110-AddA).
\item Compute ratios $R_1^{(3)} = G^{\text{M1}} / G^{\text{M3}}$ and $R_2^{(3)} = G^{\text{M2}} / G^{\text{M3}}$.
\item Check: $|R_1^{(3)} - 1| < 0.20$ and $|R_2^{(3)} - 1| < 0.20$ ?
\end{enumerate}

\textbf{Action if D3-δ fails}: Three-method triangulation is broken. Diagnose via sub-gates: does
Method 1 (TT extraction) have low SNR? Does Method 2 (Fierz-Pauli) assume a specific regularization?
Escalate to a detailed diagnostic comparison note.

\subsection*{5. Task \#128 Evaluation (2-Loop $\beta_G$)}

\subsubsection*{§5.1 Gate E1: Integration convergence}

\textbf{Criterion (from Math129 §1)}: The two-loop integral must be computed to numerical accuracy
via Richardson extrapolation in the cutoff $\Lambda_{\max}$:
$$\text{Richardson RMS}(B_H^{(2)}) < 2\%.$$

\textbf{Procedure}:
\begin{enumerate}
\item Compute $B_H^{(2)}$ at three cutoff values: $\Lambda_{\max} \in \{10 q_0, 20 q_0, 40 q_0\}$.
\item Fit Richardson extrapolation $B_H^{(2)}(\Lambda) = b_0 + b_1 / \Lambda + O(1/\Lambda^2)$.
\item Check: RMS error $< 2\%$ ?
\end{enumerate}

\textbf{Action if E1 fails}: Numerical integration is unreliable. Recommend: increase sample size,
use adaptive quadrature, or check for IR divergences in the integrand.

\subsubsection*{§5.2 Gate E2: Screening sign}

\textbf{Criterion (from Math129 §1)}: The two-loop coefficient must satisfy the physical expectation
that two-loop effects provide \emph{screening} (gravity weakens at high scales):
$$B_H^{(2)} > 0.$$

\textbf{Procedure}:
\begin{enumerate}
\item Check the sign of the extracted $B_H^{(2)}$ value.
\item If $B_H^{(2)} > 0$, screening is confirmed. \quad PASS.
\item If $B_H^{(2)} \le 0$, anti-screening or no two-loop effect. \quad FLAG but proceed to E3–E4.
\end{enumerate}

\textbf{Action}: No auto-fail; if E2 fails (anti-screening), it signals a non-standard RG structure.
Record and investigate in Round T10.

\subsubsection*{§5.3 Gate E3: Hartree validity}

\textbf{Criterion (from Math129 §1)}: The two-loop correction must be smaller than the one-loop, ensuring
Hartree is the dominant contribution:
$$B_H^{(2)} < 0.5 \times B_H^{(1)}.$$

\textbf{Procedure}:
\begin{enumerate}
\item Check: $B_H^{(2)} < 0.5 \times 0.03 = 0.015$ ?
\end{enumerate}

\textbf{Action if E3 fails}: Two-loop is as large as one-loop; Hartree perturbative control is marginal.
Recommend: functional-RG closure for Pillar 10 (not just Hartree).

\subsubsection*{§5.4 Gate E4: Fixed-point robustness}

\textbf{Criterion (from Math129 §1)}: The two-loop fixed point must not shift too dramatically from
one-loop, ensuring perturbative series convergence:
$$|g_*^{(2)} / g_*^{(1)} - 1| < 0.30.$$

\textbf{Procedure}:
\begin{enumerate}
\item Compute two-loop fixed point: $g_*^{(2)} = \text{solution to} \, \beta_G^{(2)}(g_*^{(2)}) = 0$.
\item Compute ratio: $\Delta g_* / g_*^{(1)} = |g_*^{(2)} - g_*^{(1)}| / g_*^{(1)}$.
\item Check: $\Delta g_* / g_*^{(1)} < 0.30$ ?
\end{enumerate}

\textbf{Action if E4 fails}: Fixed point shifts by $> 30\%$; series convergence questionable. Flag as
marginal control; defer final Pillar 10 status to functional-RG.

\subsubsection*{§5.5 Summary: Task \#128 pass/fail}

Task \#128 PASS: all gates (E1–E4) satisfied.
Task \#128 CONDITIONAL: E3 marginal ($B_H^{(2)} \sim 0.5 B_H$) or E4 marginal ($\Delta g_* \sim 25\%$);
proceed but note reduced confidence.
Task \#128 FAIL: E1 integration fails (RMS $> 2\%$) or E4 shows $> 50\%$ fixed-point shift.

\subsection*{6. Cross-Task Decision Logic}

\begin{enumerate}
