\section{Snapshot v2.1 fix (immediate blocker)}

\subsection{Defect}

\texttt{Codes/scripts/snapshot.ps1} step 5/8 (lines 429--438) builds a single bash command line by joining all file paths inline:
\begin{lstlisting}
$bashFiles = ($changedFiles | ForEach-Object { '"' + $_ + '"' }) -join ' '
$bashCmd = "bash Codes/scripts/sandbox_commit.sh `"$msgEsc`" $bashFiles"
& cmd /c $bashCmd
\end{lstlisting}
Windows \texttt{cmd /c} enforces an 8191-character argument-list limit (CMD.EXE legacy). With $\sim$60 files (typical post-Math353 commit including \texttt{Website/assets} mirror auto-regen), the joined command exceeds the limit and \texttt{cmd /c} returns ``명령줄이 너무 깁니다'' / ``The command line is too long''. Step 5/8 aborts with exit 1; the entire snapshot pipeline rolls back at the commit boundary.

\subsection{Patch (v2.1, this commit)}

Two-file change preserving the existing \texttt{sandbox\_commit.sh} guards (REPO\_LAYOUT \S 13 forbidden-pattern check, EOF sentinel self-integrity check, branch validation, maintainer signature):

\paragraph{Patch 1 --- \texttt{sandbox\_commit.sh} v2.1.} Argument parser extended to accept \texttt{--files-from <listfile>} in addition to inline positional arguments. The listfile is read line by line; CRLF tails are stripped via the \texttt{\$\textbackslash r} parameter expansion; blank lines are skipped. Backward-compatible: positional file arguments still work unchanged.

\paragraph{Patch 2 --- \texttt{snapshot.ps1} step 5/8.} Replaced the inline-argument build with a temp-file pattern:
\begin{lstlisting}
$stamp    = [DateTime]::UtcNow.ToString("yyyyMMddHHmmssfff")
$listFile = Join-Path $env:TEMP ("snapshot_files_" + $stamp + ".txt")
$msgFile  = Join-Path $env:TEMP ("snapshot_msg_"   + $stamp + ".txt")
$changedFiles | Out-File -FilePath $listFile -Encoding utf8 -Force
Set-Content -Path $msgFile -Value $Message -Encoding utf8 -Force -NoNewline
$bashCmd  = "bash Codes/scripts/sandbox_commit.sh -F ""$msgFile"" --files-from ""$listFile"""
& cmd /c $bashCmd
# (try/finally cleanup of temp files preserved across success and failure paths)
\end{lstlisting}
The CMD command line is now bounded by two short paths plus the bash invocation header --- well under 8 KB regardless of file count. Both temp files are removed in a \texttt{finally} block.
