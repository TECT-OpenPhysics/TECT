\subsection*{2. Atiyah--Singer Index Theorem on $\mathbb{CP}^2$}

\subsubsection*{§2.1 General statement}

For a compact oriented Riemannian manifold $M$ of dimension $4k$ (real), the **Atiyah--Singer index theorem** for the twisted Dirac operator $D_E$ is:
$$\mathrm{ind}(D_E) = \int_M \hat{A}(TM) \wedge \mathrm{ch}(E),$$
where:
\begin{itemize}
    \item $\hat{A}(TM)$ is the **A-roof genus** (or $\hat{A}$-genus) of the tangent bundle, expressed as a differential form in the Pontryagin classes.
    \item $\mathrm{ch}(E)$ is the **Chern character** of the bundle $E$, expressed as a differential form in the Chern classes.
\end{itemize}

For real dimension 4 ($k=1$), the formula becomes:
$$\mathrm{ind}(D_E) = \frac{1}{(2\pi i)^2} \int_{M} p_1(TM) \cdot c_1(E) + \text{lower-order terms}.$$

More explicitly, using the first Pontryagin class $p_1 = -\frac{1}{(2\pi)^2} \text{Tr}(R \wedge R)$ (where $R$ is the curvature 2-form of the Levi--Civita connection):
$$\mathrm{ind}(D_E) = \int_M \frac{1}{(2\pi)^2} \text{Tr}(R \wedge R) \cdot \frac{c_1(E)}{2\pi i} + \int_M c_1(E)^2.$$

\subsubsection*{§2.2 $\mathbb{CP}^2$ topology}

\noindent\textbf{Tangent bundle}: The tangent bundle of $\mathbb{CP}^2$ has:
\begin{itemize}
    \item First Pontryagin class: $p_1(T\mathbb{CP}^2) = 3 \cdot (\text{hyperplane class})^2 = 3 c_1(\mathcal{O}(1))^2$ (the standard generator of $H^4(\mathbb{CP}^2,\mathbb{Z})$).
    \item For the Fubini--Study metric, $\text{Ric} = 3 \omega_{\rm FS}$, so $p_1 = -\frac{1}{(2\pi)^2} \text{Tr}(R \wedge R)$ evaluates to a multiple of $c_1(\mathcal{O}(1))^2$.
\end{itemize}

\noindent\textbf{Chern character}: For a line bundle $E$ with $c_1(E) = 1 \cdot x$ (where $x$ is the hyperplane class), the Chern character is:
$$\mathrm{ch}(E) = e^{c_1(E)} = 1 + c_1(E) + \frac{1}{2} c_1(E)^2 + \cdots = 1 + x + \frac{x^2}{2} + \text{higher}.$$

The degree-4 part is $\frac{c_1(E)^2}{2} = \frac{x^2}{2}$.

\subsubsection*{§3.2 Holomorphic sections and vector bundles}

A more direct approach: on a complex manifold $M$, the **Dolbeault cohomology** $H^{0,q}(M, E)$ computes the space of $\bar{\partial}$-closed $(0,q)$-forms with values in a holomorphic vector bundle $E$.

For $\mathbb{CP}^2$ and a line bundle $\mathcal{O}(1)$:
$$\dim H^0(\mathbb{CP}^2, \mathcal{O}(1)) = 3 \quad \text{(holomorphic sections: linear forms in three variables)}.$$
$$\dim H^1(\mathbb{CP}^2, \mathcal{O}(1)) = 0, \quad \dim H^2(\mathbb{CP}^2, \mathcal{O}(1)) = 0.$$

The **Euler characteristic** (index for the Dolbeault complex) is:
$$\chi(\mathbb{CP}^2, \mathcal{O}(1)) = \dim H^0 - \dim H^1 + \dim H^2 = 3 - 0 + 0 = 3.$$

For an $\mathrm{SU}(5)$-bundle with associated vector bundle $E$ of rank 21, the index of the twisted Dirac operator is related to the **total Chern character** of $E$.
