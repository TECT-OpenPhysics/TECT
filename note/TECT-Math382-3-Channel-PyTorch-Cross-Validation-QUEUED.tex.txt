\subsection*{1. The reduction being validated}

The Math376 v3 framework (and all derived notes Math378-394) operate
on a single real scalar field $\Psi_{\rm real} \in \mathbb{R}^{N\times N\times N}$
obtained by projecting the production-format $(3, N, N, N)$ complex
field onto the family-locked direction $\mathbf z_0 = (1,1,1)/\sqrt 3$:
\begin{equation}
\Psi_{\rm real}(\mathbf r) = \mathrm{Re}\left[\mathbf z_0 \cdot
\boldsymbol\Psi(\mathbf r)\right] = \mathrm{Re}\left[\sum_{a=1}^{3}
z_{0,a}\,\Psi_a(\mathbf r)\right].
\label{eq:projection}
\end{equation}

The canonical Brazovskii free energy used in Math376 v3 is
\begin{equation}
\mathcal{F}_{\rm 1ch}[\Psi_{\rm real}] = \int d^3 r \left[
\tfrac{r}{2}\Psi_{\rm real}^2 + \tfrac{Z}{2}|\nabla\Psi_{\rm real}|^2
+ \tfrac{Y}{2}|\Delta\Psi_{\rm real}|^2
+ \tfrac{\lambda}{2}\Psi_{\rm real}^4 + \tfrac{\gamma}{3}\Psi_{\rm real}^6
\right]
\label{eq:F-1ch}
\end{equation}

The production 3-channel \texttt{shell\_free\_energy} computes a
DIFFERENT functional that includes additional family/lock/classII
penalty terms:
\begin{equation}
\mathcal{F}_{\rm 3ch}[\boldsymbol\Psi] = \mathcal{F}_{\rm canonical-3ch}
[\boldsymbol\Psi] + F_{\rm fam} + F_{\rm lock} + F_{\rm classII}
\label{eq:F-3ch}
\end{equation}
(see \texttt{shell\_free\_energy} lines 553--599 of
\texttt{real\_backend\_pt\_bcc\_mixed\_v3.py}).

Math376 v3's reduction \eqref{eq:projection} + \eqref{eq:F-1ch}
neglects $F_{\rm fam}, F_{\rm lock}, F_{\rm classII}$ on the
assumption that they vanish at the family-locked equilibrium
state. \textbf{This assumption is plausible but unverified.}

\subsection*{2. Pre-registered PASS criterion}

For a pre-converged BCC state at $\mu^2 = -1$, $N = 32$ (e.g.,
\texttt{Runs/math376/math376\_converged\_l\_bfgs\_b\_Psi\_BCC\_N32\_phaseZ.npy}):

\begin{equation}
\boxed{\;
\frac{|\mathcal{F}_{\rm 3ch}[\boldsymbol\Psi^{\rm 3ch-promoted}] -
\mathcal{F}_{\rm 1ch}[\Psi_{\rm real}^{\rm Math376}]|}{|\mathcal{F}_{\rm 1ch}|}
< 10^{-3}
\;}
\end{equation}

\textbf{PASS}: relative agreement to $10^{-3}$ certifies the
single-channel reduction is faithful within $\sim 0.1\%$. All
Math376-394 cascade results carry over with this $\sim 0.1\%$
systematic uncertainty.

\textbf{INDETERMINATE}: relative disagreement in $[10^{-3}, 10^{-1}]$.
Reduced model is approximately faithful but not high-precision.
Math376-394 results carry $\sim 1\%$--$10\%$ uncertainty; some
verdict-magnitudes might shift but qualitative conclusions stand.

\textbf{FAIL}: relative disagreement $> 10^{-1}$. Reduced model is
substantially inaccurate. Math376-394 results require complete
re-verification under the full 3-channel framework. This would be a
MAJOR setback (would need to revisit Math378, 380, 390 PASS verdicts).

\subsection*{3. Implementation script (queued)}

\textbf{Filename}: \texttt{Codes/supplementary/Math382\_3channel\_crossvalidation.py}.

\textbf{Pseudocode} (operator to write/adapt; requires PyTorch on
operator PC):

\begin{verbatim}
import numpy as np
import sys; sys.path.insert(0, "Codes/pde")
from real_backend_pt_bcc_mixed_v3 import shell_free_energy

# Step 1: load Math376 v3 single-channel converged state (.npy 3D real)
psi_1ch = np.load("Runs/math376/math376_converged_l_bfgs_b_Psi_BCC_N32_phaseZ.npy")
# psi_1ch shape: (32, 32, 32) real
N = psi_1ch.shape[0]

# Step 2: promote single-channel real to 3-channel complex via z0
z0 = np.array([1.0, 1.0, 1.0]) / np.sqrt(3.0)
psi_3ch = np.zeros((3, N, N, N), dtype=np.complex128)
for a in range(3):
    psi_3ch[a] = z0[a] * psi_1ch

# Step 3: compute F under each formulation
# (a) Math376 v3 canonical 1ch:
import importlib.util
spec = importlib.util.spec_from_file_location(
    "m376", "Codes/supplementary/Math376_production_state_hessian.py")
m376 = importlib.util.module_from_spec(spec); spec.loader.exec_module(m376)
L = 62.20036; q0 = 0.6801747616
grid = m376.setup_kspace(N, L)
params = {"r": -1 + q0**4, "Z": -2*q0**2, "Y": 1.0,
          "lam": -0.43, "gam": 1.62, "q0": q0, "mu2": -1.0}
F_1ch = m376.free_energy_canonical(psi_1ch, grid, params)

# (b) production 3-channel shell_free_energy
prod_params = {**params, "Lx": L, "Ly": L, "Lz": L,
               "alpha_X": 0.3, "beta_X": 0.25, "M_X": 2.0,
               "cJJ": 0.2, "cJK": 0.1, "cKK": 0.15,
               "k_lock": 0.15, "eta_shell": 0.0,
               "family_masses": [0.0, 0.03, 0.07],
               "z0": [1.0, 1.0, 1.0]}
F_3ch = shell_free_energy(psi_3ch, prod_params)

# Step 4: report
rel = abs(F_3ch - F_1ch) / abs(F_1ch)
print(f"F_1ch = {F_1ch:+.6e}")
print(f"F_3ch = {F_3ch:+.6e}")
print(f"|F_3ch - F_1ch| / |F_1ch| = {rel:.6e}")
verdict = "PASS" if rel < 1e-3 else ("INDETERMINATE" if rel < 1e-1 else "FAIL")
print(f"Verdict: {verdict}")
\end{verbatim}

\subsection*{6. Quantitative sanity check (CLAUDE.md \S 6.3.4)}

\textbf{Magnitude expectation}: family-locking penalty
$F_{\rm lock} \sim (1/k_{\rm lock}) \cdot |\boldsymbol\Psi -
z_0 \cdot \Psi_{\rm scalar}|^2$. For a perfectly family-locked
state $\boldsymbol\Psi = z_0 \cdot \Psi_{\rm scalar}$, the bracket
is zero and $F_{\rm lock} = 0$. For a near-locked state with
deviation $\epsilon$, $F_{\rm lock} \sim \epsilon^2 / k_{\rm lock}$.
With $k_{\rm lock} = 0.15$ and $\epsilon \sim e^{-1/k_{\rm lock}}
\sim 10^{-3}$, $F_{\rm lock} \sim 10^{-6}$. Compared to
$|F_{\rm canonical}| \sim 10^4$ at $N = 32$ deep BCC, ratio
$\sim 10^{-10}$ — well below PASS criterion $10^{-3}$.

\textbf{$F_{\rm classII}$ contribution}: order $\alpha_X^2 / M_X^2
\sim 0.09/4 = 0.02$ (locked params); only matters if $|J|$ or
$|K|$ tensors are non-zero, which they are at the locked equilibrium.
Numerical evaluation required for tight bound.

\textbf{Conclusion}: expected outcome PASS at $10^{-3}$ to $10^{-6}$
agreement, dominated by classII residuals. Math382 PASS very likely;
INDETERMINATE possible; FAIL would be surprising.
