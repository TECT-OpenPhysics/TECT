being theoretically correct and the parameter lock being physically
motivated. The action FORM is verified; the parameter VALUES depend

\subsection*{4. The honest scope of the methodology failure}

The TECT cascade pattern (over ~6 months, Math1 to Math396):
\begin{enumerate}
\item Brazovskii free energy assumed as canonical action (F1).
\item BCC ansatz assumed for explicit calculation (F4 implicit).
\item Each pillar derived UNDER BCC assumption (F5).
\item Numerical verification of BCC LOCAL stability (Math82 phase-Z,
  Math373-380 cascade) was performed — but \emph{LOCAL stability ≠
  GLOBAL min}.
\item Math383 §6 (2026-05-11) is the FIRST direct test of BCC
  global minimality vs competitors. It suggests BCC is NOT global.
\end{enumerate}

\textbf{What was missing}: a foundation-first verification protocol
that would have tested F4 (BCC global min) BEFORE building F5
(pillar derivations). Such a test would have been a 30-minute
operator-PC computation; it was not performed for ~6 months.

\textbf{Structural lesson}: The cascade pattern ``develop theory
upward, then audit'' is fragile. The proper pattern is
``verify foundation, then develop''. Math398 enacts this pattern
retroactively.
