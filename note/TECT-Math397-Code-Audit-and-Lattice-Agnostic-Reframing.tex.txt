\textbf{(1) Lattice wavevectors} (\texttt{lattice\_wavevectors}):
\begin{itemize}

\textbf{(2) Seed amplitude formula} (\texttt{deep\_amplitude\_for\_lattice}):

A = \sqrt{\phi_0^2 / (n_{\rm modes}/2)}
\end{equation}
where $\phi_0^2 = \langle\Psi^2\rangle = (n_{\rm modes}/2) A^2$ is
derived from cosine-orthogonality. ✓

in Math376):

\end{equation}
computed by direct integration on the FFT grid; \emph{does NOT
depend on $K_4, K_6$ structure factors}. ✓

\textbf{(4) Convention issue identified}: $K_4, K_6$ values used
in the script for non-BCC structures (FCC: $1.5/3.5$, HEX: $2.0/5.0$,
lamellar: $3.0/10.0$) are from Leibler (1980) Table II. TECT
canonical convention gives different values:
\begin{itemize}
\item Lamellar TECT: $K_4 = 1.5, K_6 = 2.5$ (script used $3.0, 10.0$)
\item HEX TECT: not derived in this audit; needs Math397-AddA.
\item FCC TECT: not derived in this audit; needs Math397-AddA.
\end{itemize}

\textbf{Impact of convention error}: $K_4, K_6$ are used \emph{only}
in the seed amplitude calculation (`deep\_amplitude\_for\_lattice`).
Wrong $K$ values give wrong seed amplitude, but L-BFGS-B converges
the seed to the actual stationary point of the canonical free
energy regardless. The CONVERGED $F$ values are correct. The
COMPARISON among converged $F$ values is therefore valid.

\textbf{Verdict}: \textbf{Code is correct for the F comparison.
The Math383 §3-4 1-mode analytical predictions for non-BCC
structures used wrong $K$ convention (Math397-AddA fix queued),
but the Math383 §6 numerical result LAMELLAR < BCC at both
operating points STANDS.}

Cross-check at lamellar with TECT-correct $K$:
\begin{itemize}
\item 1-mode prediction at $\mu^2 = +0.005$: $\phi_0^2_{\rm lamellar}
  = 0.155$, $F/V = -2.33\times 10^{-3}$.
\item Numerical: $\phi_0^2 = 0.165$, $F/V = -2.39\times 10^{-3}$.
\item Agreement: $3\%$. ✓
\end{itemize}

Cross-check at BCC with TECT-canonical $K_4 = 1.0, K_6 = 2.5$:
\begin{itemize}
\item 1-mode prediction at $\mu^2 = +0.005$: $\phi_0^2_{\rm BCC} = 0.10$,
  $F/V = -5.5\times 10^{-4}$.
\item Numerical: $\phi_0^2 = 0.142$, $F/V = -1.80\times 10^{-3}$.
\item Disagreement: factor $3.3\times$ in F.
\end{itemize}

The BCC numerical converged to $\phi_0^2 = 0.142$ ($42\%$ above
1-mode prediction $0.10$), suggesting multi-mode corrections at the
canonical operating point are LARGE for BCC. This is a separate
physical phenomenon — BCC's convergence to a state with extra
non-shell modes — not a code bug.

\textbf{Even with TECT-correct $K$ analytics}: lamellar 1-mode
$F/V = -2.33\times 10^{-3}$ remains LOWER than BCC 1-mode
$F/V = -5.5\times 10^{-4}$. The lattice ordering lamellar < BCC
holds analytically AND numerically.
