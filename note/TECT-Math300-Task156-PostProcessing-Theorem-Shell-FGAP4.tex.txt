\begin{equation}

\end{equation}
i.e., the eigenvalue is converted to a mass-squared via the

\begin{equation}
\mu_{\rm defect} = \underbrace{a_{\rm BCC}^{-1}\cdot\sqrt{\hbar c}}_{
   K_{\rm tr}\,\sim\,10^{19}\text{ GeV}}\cdot \sqrt{\hat\lambda_{\min}}.

The constant $K_{\rm tr}$ is the Planck-scale-defect translation
factor; for $a_{\rm BCC} = 4\sqrt{\pi}\,\ell_P$, $\hbar c = 0.1973\times

\begin{equation}

\;\approx\;1.22\times 10^{19}\,\mathrm{GeV}.
\end{equation}

correctly recovered.]

\section{Theorem statement and verdict-chain proof}
\label{sec:theorem}

\begin{theorem}[Verdict-chain determinism]
\label{thm:chain}
The verdict-emission chain
\[
\hat\lambda_{\min}\;\longrightarrow\;\mu_{\rm defect}\;\longrightarrow\;
\{\text{PASS, FAIL, DEFER}\}
\]
is deterministic given (i) the Task \#156 HYM solver output, (ii)
the calibrated $K_{\rm tr}$ constant, (iii) the F-GAP4 window
$[10^{13}, 10^{17}]$ GeV, and (iv) the convergence/precision
auxiliary gates listed in Definition \ref{def:verdict} DEFER branch.
The verdict is unambiguous (exactly one of PASS/FAIL/DEFER fires).
\end{theorem}

\begin{proof}
The translation $\hat\lambda_{\min}\to\mu_{\rm defect} = K_{\rm tr}
\sqrt{\hat\lambda_{\min}}$ is a single-valued monotone function on
the positive real axis. The interval test
$\hat\lambda_{\min}\in[\lambda_{\rm low}, \lambda_{\rm high}]$
partitions the positive axis into three regions: below, in, above.
The DEFER branch is gated by independent solver-level conditions
(convergence, grid resolution, numerical precision) which are
boolean and binary. The decision tree is:
\begin{enumerate}
\item Check DEFER conditions first (any one fires → DEFER, terminate).
\item Else, check $\hat\lambda_{\min} \in [\lambda_{\rm low},
\lambda_{\rm high}]$: yes → PASS, no → FAIL.
\end{enumerate}
Exactly one outcome is emitted; the chain is functional, not
relational. \qed
\end{proof}
