\subsection*{1. Route A: ℏ$_{\rm Fock}$ from Canonical Commutation Relations}

\subsubsection*{§1.1 Fock-space field operator}

From Math140, the quantum field operator is defined via Fock decomposition:
\begin{equation}
\hat{\Psi}(\mathbf{x}, t) = \sum_\mathbf{k} \sqrt{\frac{\hbar}{2 \omega_\mathbf{k} V}}
  \left[ a_\mathbf{k} \phi_\mathbf{k}(\mathbf{x}) e^{-i \omega_\mathbf{k} t}
    + a_\mathbf{k}^\dagger \phi_\mathbf{k}^*(\mathbf{x}) e^{i \omega_\mathbf{k} t} \right],
\label{eq:fock-field}
\end{equation}
where:
\begin{itemize}
\item $a_\mathbf{k}, a_\mathbf{k}^\dagger$ = annihilation/creation operators for shell mode $\mathbf{k}$.
\item $\phi_\mathbf{k}(\mathbf{x})$ = shell-mode eigenfunction (Math139).
\item $\omega_\mathbf{k}$ = dispersion relation: $\omega_\mathbf{k} = c_T |\mathbf{k}|$ (to leading order, from Pillar 2, Math39).
\item $V$ = spatial volume.
\item $\hbar$ = the quantum action scale (to be determined).
\end{itemize}

The conjugate momentum operator is:
\begin{equation}
\hat{\Pi}_\Psi(\mathbf{x}, t) = \rho_{\rm cond} \, \frac{\partial \hat{\Psi}}{\partial t}.
\label{eq:fock-momenta}
\end{equation}

\subsubsection*{§1.2 Equal-time canonical commutation relation}

From Math141, the equal-time CCR is:
\begin{equation}
[\hat{\Psi}(\mathbf{x}, t), \hat{\Pi}_\Psi(\mathbf{x}', t)] = i \hbar \, \delta^3(\mathbf{x} - \mathbf{x}').
\label{eq:ccr}
\end{equation}

This is the \emph{defining relation} for the quantum field operator. The constant $\hbar$ is the Fock-space normalization scale.

\subsubsection*{§1.3 Explicit vacuum-to-one-particle matrix element}

To extract the numerical value of $\hbar$, compute the one-particle creation matrix element:
\begin{equation}
\langle 1_\mathbf{k} | a_\mathbf{k}^\dagger | 0 \rangle = \langle 0 | a_\mathbf{k} a_\mathbf{k}^\dagger | 0 \rangle^{1/2}.
\label{eq:vac-to-one}
\end{equation}

Using $[a_\mathbf{k}, a_\mathbf{k}^\dagger] = 1$ (canonical algebra), we have:
\begin{equation}
a_\mathbf{k} a_\mathbf{k}^\dagger = a_\mathbf{k}^\dagger a_\mathbf{k} + 1,
\end{equation}
so:
\begin{equation}
\langle 0 | a_\mathbf{k} a_\mathbf{k}^\dagger | 0 \rangle = 1.
\end{equation}

Thus the one-particle state is properly normalized: $\langle 1_\mathbf{k} | 1_\mathbf{k} \rangle = 1$.

\subsubsection*{§1.4 Zero-point energy and Fock normalization}

The zero-point energy of the vacuum is:
\begin{equation}
E_{\text{vac}} = \sum_\mathbf{k} \frac{1}{2} \hbar \omega_\mathbf{k}.
\label{eq:zpne}
\end{equation}

The Fock-space normalization requires that the zero-point energy scales as:
\begin{equation}
E_{\text{vac}} \sim \hbar \times (\text{characteristic frequency}).
\label{eq:zpne-scaling}
\end{equation}

For the BCC condensate, the characteristic frequency is the shell-mode cutoff $\omega_{\max} \sim c_T / a_{\rm BCC}$.

\subsubsection*{§1.5 Numerical value of ℏ$_{\rm Fock}$}

Combine the Fock normalization with the shell-mode structure. The natural scale is set by the product:
\begin{equation}
\hbar_{\rm Fock} = \rho_{\rm cond} \times c_T \times a_{\rm BCC}^3.
\label{eq:hbar-fock}
\end{equation}

This is precisely the formula from Math98 (phase-transition origin of $\hbar$).

Substituting the v2.4 operating point (from Math55–56 continuation):
\begin{itemize}

\item $c_T = c$ (from Pillar 2, Math110-AddH, universality-class identification).

We get:

  = \frac{c^5 a_{\rm BCC}}{16\pi G}.
\label{eq:hbar-fock-explicit}
\end{equation}

\subsubsection*{§2.1 Stress-energy tensor of the condensate}

The Einstein field equations (Pillar 3) are:
\begin{equation}
G_{\mu\nu} = 8\pi G \, T_{\mu\nu},
\label{eq:einstein}
\end{equation}

For the BCC condensate, the stress-energy tensor is:
\begin{equation}
T_{\mu\nu}^{\rm cond} = (\rho_{\rm cond} + P_{\rm cond}/c^2) u^\mu u^\nu - P_{\rm cond} g_{\mu\nu},
\label{eq:tee-cond}
\end{equation}
where:
\begin{itemize}
\item $\rho_{\rm cond}$ = energy density.
\item $P_{\rm cond}$ = pressure.
\item $u^\mu$ = four-velocity (TECT at rest in the lab frame: $u^\mu = (1, 0, 0, 0)$ in natural units $c=1$).
\end{itemize}

\begin{itemize}
\item $Y$ = shape-factor coefficient (from BCC geometry, Math41).
